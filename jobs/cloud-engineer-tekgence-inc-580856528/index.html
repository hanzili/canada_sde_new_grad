Role: Developer

Location: TORONTO

Develop FHIR-based APIs and interfaces for healthcare data. Implement secure data ex-change between systems (EHRs, devices).Utilize Azure services Azure Functions, API Management, Service Bus, Event Hubs, MedTech. Work with healthcare standards (HL7, FHIR) and security protocols (OAuth 2.0, OIDC).Perform data mapping, transformation, and integration. Write clean, well-tested code (C, Python often) and use version control (Git).Troubleshoot integration issues and optimize workflows. An Azure FHIR Developer designs, builds, and integrates healthcare applications using the FHIR standard on Microsoft Azure, focusing on secure data exchange (APIs, HL7), using Azure services like Functions APIM Event Hubs, with skills in C, .NET Core, RESTful APIs, data mapping, and Git, ensuring compliance and interoperability for better patient data access and system integration.
