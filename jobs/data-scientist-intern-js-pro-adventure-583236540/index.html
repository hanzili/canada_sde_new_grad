Full Description

**Overview** We are seeking a highly skilled Data Scientist to join our innovative analytics team. The ideal candidate will leverage advanced data analysis, machine learning, and AI techniques to extract actionable insights from complex datasets. This role offers an exciting opportunity to work with cutting-edge technologies such as AWS, TensorFlow, Hadoop, Spark, and natural language processing to drive strategic decision-making and optimize business processes. The position is full-time and involves collaboration across multiple departments to develop scalable data solutions that support organizational growth.

**Duties**

- Design and implement scalable data models and algorithms using Python, R, Java, and SQL to solve complex business problems. - Develop machine learning and AI models utilizing frameworks like TensorFlow, SAS, and Spark for predictive analytics and automation. - Conduct data mining and big data analysis to identify trends, patterns, and insights that inform strategic initiatives. - Build and maintain robust database architectures with expertise in database design, ensuring efficient data storage and retrieval. - Perform natural language processing (NLP) tasks to analyze unstructured text data for sentiment analysis, chatbots, or other applications. - Collaborate with cross-functional teams to translate business requirements into technical specifications and actionable data solutions. - Utilize cloud platforms such as AWS for deploying scalable machine learning models and managing large datasets. - Develop scripts using Bash (Unix shell) and VBA for automation of routine tasks and report generation. - Stay current with emerging trends in analytics, AI, and big data technologies to continuously improve analytical capabilities.

**Experience**

- Proven experience as a Data Scientist or in a similar role with a strong portfolio of projects involving machine learning, AI, or big data analytics. - Proficiency in programming languages including Python, R, Java, C, and SQL. - Hands-on experience with cloud services such as AWS for deploying machine learning models at scale. - Familiarity with big data frameworks like Hadoop, Spark, and tools such as SAS for large-scale data processing. - Expertise in database design principles along with experience working with relational and NoSQL databases. - Strong understanding of data mining techniques, natural language processing (NLP), and analytics methodologies. - Knowledge of tools like TensorFlow for deep learning applications; experience with Hadoop ecosystem is a plus. - Ability to interpret complex datasets into clear insights that support strategic decision-making in a fast-paced environment. This position offers an engaging environment where innovation meets practical application of advanced analytics technologies to solve real-world problems.
