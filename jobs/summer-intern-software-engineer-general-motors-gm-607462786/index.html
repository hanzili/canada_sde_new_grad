About the Team

The Motion Estimation Team plays a key role in the software development of AV/ADAS systems at General Motors. This team is responsible for delivering integrated and tested software for various GM features, including Super Cruise, Adaptive Cruise Control, Lane Centering, Lane Keep Assist and others.

About The Role :

This role will be working on software for ADAS Motion Estimation Software team. Areas of focus may include implementing, testing, and/or integration.

What You’ll Do:

Architect, design, implement, integrate and test robust embedded software solutions for ADAS systems. This includes the following

Write and maintain code in C, Simulink and C++ (preferred)

Analyze and improve reliability and efficiency.

Create and maintain technical documentation

Perform unit testing, integration testing, and system testing.

Preferred Qualifications :

Experience in state estimation domain

Embedded software experience/knowledge

Software Product Line experience/knowledge

Minimum Qualifications:

Working towards Bachelors, Master degree in Computer Science or Engineering (Electrical, Computer, Mechatronics, System Design, Mechanical, Chemical or equivalent)

C software skills.
