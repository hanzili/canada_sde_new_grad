## **Our team ...**

The internship will take place in Engineering, in the Simulation team. The Simulation group develops the flight simulation model for all Bombardier aircraft as well as development projects. The group also develops its own tools for development, documentation, and validation purposes. Finally, the group is responsible for developing various platforms where the model is used in combination with aircraft hardware (for instance, fixed base simulators).The simulation model is an essential engineering tool which is used at every step of an aircraft life, including design, flight test, crew training and in-service support.

The goal of this internship is to continue the development of a software allowing to visualize simulation results as well as flight test data using AI.

### **What are your contributions to the team?**

- Literature review on exciting AI methods in simulation industry. - Plan and perform software development in C++ and Python. - Develop and update the software requirements. - Develop tests to validate the software functionalities. - Understand the link between the software to develop and the work involved in developing a flight simulation model. - Organize meetings to review the software with users to obtain feedback. - Develop skills in technical writing (to document the software for programmers and for users). - Participate to team meetings and be exposed to the tasks performed by the Simulation group. - Develop other tools for the Simulation group.

### **How to thrive in this role?**

- You are enrolled in a Software, Electrical, Mechanical or Aerospace program. - You master Python and C++ programming. - Asset: you have already programmed a project in C++ - You know how to define efficient algorithms and to write easy-to-understand code. - You are organized and capable of planning the code development to meet the user requirements. - You are meticulous and capable of defining validation tests. - You are autonomous and able to execute a task under minimum supervision. - Asset: you know how to program in Matlab. - You speak and write in English.
