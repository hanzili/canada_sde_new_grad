À propos du projet

Notre mission

Chez Vocality, nous développons des outils de communication assistée pour les personnes ayant perdu l’usage de la parole. Alimentée par l’IA, notre application apprend le style de chaque utilisateur et offre une saisie intelligente, des prédictions en temps réel et la synthèse vocale pour redonner autonomie et fluidité à la communication.

Sur quoi vous travaillerez

  • Construire et perfectionner les composants UI du client (boutons, panneaux, champs de saisie, composants clavier)
  • Implémenter des mises en page responsives utilisant des composants personnalisés
  • Créer des interfaces accessibles pour les utilisateurs ayant des difficultés motrices
  • Travailler avec l'architecture MVVM et l'injection de dépendances
  • Intégrer le frontend avec les endpoints backend FastAPI

Tech Stack

Framework: Avalonia (cross-platform .NET UI framework)

Langage: C#, AXAML (XAML d’Avalonia)

APIs: Intégration RESTful avec backend FastAPI

Exigences

  • Actuellement inscrit comme étudiant en informatique, i ngénierie ou domaine connexe
  • Connaissances intermédiaires de C#
  • Compréhension des principes UI/UX
  • Familiarité avec les modèles MVVM ou MVC
  • Expérience avec Git/GitHub
  • Grande attention aux détails et considérations d'accessibilité

Atouts

  • Expérience avec Avalonia, WPF ou frameworks similaires basés sur XAML
  • Expérience en développement d'applications desktop
  • Connaissance de .NET
  • Expérience avec les technologies d'assistance ou design axé sur l'accessibilité

Ce que nous offrons

  • Horaires flexibles adaptés à votre calendrier académique
  • Expérience pratique dans la construction de technologies d'assistance
  • Mentorat et revue de code pour développer vos compétences
  • Opportunité d'avoir un impact social positif
  • Opportunité de progresser au sein de l'entreprise

About the Project

Our mission

At Vocality, we build assistive communication tools for people who have lost the ability to speak. Powered by AI, our application learns each user’s communication style and delivers smart typing, real-time predictions, and text-to-speech to restore autonomy and fluid, effortless communication.

What You'll Be Working On

  • Building and refining UI components for the client (buttons, panels, input fields, keyboard components)
  • Implementing responsive layouts using custom components
  • Creating accessible, user-friendly interfaces for users with motor difficulties
  • Working with MVVM architecture and dependency injection
  • Integrating frontend with FastAPI backend endpoints

Tech Stack

Framework: Avalonia (cross-platform .NET UI framework)

Language: C#, AXAML (Avalonia's XAML)

APIs: RESTful integration with FastAPI backend

Must Have

  • Currently enrolled as a student in Computer Science, Engineering or related field
  • C# Knowledge
  • Understanding UI/UX principles
  • Familiar with MVVM or MVC principles
  • Attention to detail and accessibility considerations
  • GitHub experience

Nice To Have

  • Experience with Avalonia, WPF, or similar XAML-based frameworks
  • Desktop application development experience
  • Knowledge of .NET
  • Experience with assistive technology or accessibility-focused design

What we offer

  • Flexible hours that work around your academic schedule
  • Real-world experience building meaningful assistive technology
  • Mentorship and code review to grow your skills
  • Opportunity to make a positive social impact
  • Opportunity to progress in the company

How to Apply

  • Resume/CV by email only

: contact@vocality.ca - GitHub profile or code samples - Brief explanation of why you're interested in assistive technology - Your availability (hours per week)
