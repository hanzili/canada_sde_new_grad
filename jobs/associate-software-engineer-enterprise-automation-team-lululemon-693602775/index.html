About lululemon

Full Description

Requirements Description & Requirements Who we are lululemon is an innovative performance apparel company for yoga, running, training, and other athletic pursuits. Setting the bar in technical fabrics and functional design, we create transformational products and experiences that support people in moving, growing, connecting, and being well. We owe our success to our innovative product, emphasis on stores, commitment to our people, and the incredible connections we make in every community we're in. As a company, we focus on creating positive change to build a healthier, thriving future. In particular, that includes creating an equitable, inclusive and growth-focused environment for our people.

About This Team The Enterprise Automation team is focused on elevating the organization by removing repetitive, manual work and enabling teams to operate more efficiently through intelligent automation. We deliver scalable, secure, and well‑governed automation solutions across the enterprise, primarily using UiPath and the Microsoft Power Platform, while actively evolving toward agentic automation and AI‑assisted workflows.

Our team works closely with business partners, architects, and platform teams to design, build, and support automations that span systems, departments, and geographies. In addition to traditional RPA, we are investing in Agentic Automation, including early use of UiPath AI/Agent capabilities and Copilot Studio, with a strong emphasis on responsible enablement, governance, and real business value.

Core Responsibilities As an Associate Software Engineer, you will develop software features and components with guidance from senior engineers, writing functional code for assigned work including API endpoints, database queries, and UI components while building technical proficiency in the team’s technology stack and development practices. You will participate in design discussions to understand technical approaches, conduct code reviews for peers providing constructive feedback, and contribute to feature delivery from requirements clarification through production deployment and post-release monitoring. You will collaborate effectively with product managers and designers on feature requirements, debug and resolve issues in development and production environments, and contribute to team knowledge sharing through documentation and technical discussions. This role may require availability outside of standard business hours, including on‑call support, based on business needs.

Select Responsibilities Include

  • Design and implement software features end-to-end including APIs, business logic, data models, and user interfaces within established architectural patterns
  • Write clean, maintainable code following established coding standards and design patterns
  • Conduct code reviews providing constructive feedback on code quality, design, and security
  • Write comprehensive automated tests including unit, integration, and end-to-end tests
  • Debug and resolve defects and production issues including root cause analysis and implementing fixes

Qualifications

  • Bachelor’s degree in Computer Science, Software Engineering, or related technical field, or equivalent experience
  • 2-4 years of software development experience building and maintaining production applications, or equivalent
  • Experience writing clean, maintainable code following established coding standards; familiarity with design patterns and code review practices
  • Proven ability to decompose moderately complex problems into software solutions; understanding of how code changes can impact related components
  • Experience implementing API integrations following REST principles; familiarity with authentication mechanisms and established integration patterns
  • Foundational understanding of RPA concepts and automation use cases.

Additional Notes Authorization to work in Canada is required for this role.
