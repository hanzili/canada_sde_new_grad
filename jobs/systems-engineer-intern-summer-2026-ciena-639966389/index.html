Software Engineering Internship

Systems Engineer Intern (Summer 2026)

at Ciena

📍 Location Ottawa, Ontario
📋 Details Intern · 8mo · $25.00-32.00/hr · Summer 2026
📅 Posted Mar 09, 2026

About the Role

Develop GUI and backend for CMIS data

About Ciena

Global network equipment leader; strong engineering culture

Full Description

As the global leader in high-speed connectivity, Ciena is committed to a people-first approach. Our teams enjoy a culture focused on prioritizing a flexible work environment that empowers individual growth, well-being, and belonging. We’re a technology company that leads with our humanity—driving our business priorities alongside meaningful social, community, and societal impact.

The Opportunity This eight‑month work term (May–December 2026) offers an opportunity to contribute to next‑generation networking technologies that enable high‑speed connectivity across data centers and optical transport systems. This role supports the development of a software tool that transforms complex raw Common Management Interface Specification (CMIS) data into clear, meaningful insights used across engineering and customer‑facing teams.

How You Will Make An Impact

  • Support development of a graphical user interface (GUI) and backend components that automate the translation and interpretation of raw CMIS Electrically Erasable Programmable Read‑Only Memory (EEPROM) data
  • Build logic that adapts to varying CMIS data formats from third‑party routers and platforms
  • Contribute to lab validation activities, including automated software test execution
  • Participate in proof‑of‑concept demonstrations and solution evaluations for internal stakeholders
  • Collaborate with optical, routing, and software engineers to integrate Ciena transmitter technology within diverse platforms
  • Document technical findings, workflows, and data‑parsing rules to support ongoing tool development
  • Investigate and troubleshoot issues related to data formats, hardware behavior, or software logic

The Must Haves

  • Enrollment in a Bachelor of Engineering, Computer Science, or equivalent degree program (“in progress”)
  • Proficiency with Python, JavaScript, and QT
  • Knowledge of Web UI concepts including HTML5, CSS, YANG, XML, and JSON
  • Understanding of low‑level hardware concepts such as bits, bytes, data types, and memory registers
  • Ability to investigate and troubleshoot technical issues methodically
  • Strong verbal and written communication skills
  • Strong organizational skills and attention to detail

Nice To Haves

  • Experience with optical lab instrumentation or optical networking technologies
  • Familiarity with software APIs and hardware/software integration workflows
  • Exposure to optical networking terminology and standards as used across the industry
  • Experience contributing to software tools used in engineering or lab environments
  • Knowledge of automation frameworks or scripting for test environments
