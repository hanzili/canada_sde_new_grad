Full Description

**Job Description** The SAR 7705 Test team is responsible for designing, developing, and executing advanced test scripts to ensure that software features conform to the requirements defined by Product Management. These scripts validate both the conformance and functionality of implementations delivered by Software Engineering. Our testing scope is broad and dynamic, covering areas such as data path performance, control plane protocols, and Layer 3 technologies, including IP/MPLS, OSPF, IS-IS, and BGP. We also focus on Operations, Administration, and Maintenance (OAM) protocols like CFM, EFM, TWAMP, and Performance Monitoring. Additional areas of expertise include timing and synchronization technologies such as PTP, NTP, SyncE, SETS, BITS, and GNSS; security and quantum-ready encryption solutions like IPSec, MacSec, Network Group Encryption, Anysec, OpenSSL, FIPS, and NIAPS; and physical layer testing involving Ethernet, MAC, SerDes, 1G/10G Optical Transceivers, and legacy interfaces such as serial, T1/E1, C37.94, and voice bridges. Our responsibilities also extend to testing Layer 2 and Layer 3 VPNs, as well as legacy services including CESoPSN, Serial, TDM, Ethernet PW, and IP PW. **How You Will Contribute And What You Will Learn**

- Develop, modify, and execute software test plans to ensure product quality and performance. - Analyze and write test standards and procedures, maintaining thorough documentation of test results. - Review test outcomes to verify functionality and recommend corrective actions as needed. - Collaborate with top engineers to apply academic knowledge to real-world challenges and drive innovation. - Assist in refining customer requirements where appropriate to align testing with user needs. - Create automated test cases to validate new hardware and software features in a Linux-based regression environment. - Troubleshoot issues and work closely with testers and developers to investigate and resolve bugs. - Learn and apply industry best practices in software development, configuration management, and quality assurance automation.
