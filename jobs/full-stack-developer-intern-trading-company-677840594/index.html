About Trading Company

Full Description

Overview
We are looking for a motivated Full Stack Developer Intern to join our software development team working mainly in process automation and AI. The ideal candidate will have a background in both front-end and back-end development, with knowledge in building scalable, efficient applications. This role offers the opportunity to work on innovative projects utilizing a wide range of technologies, including web development, cloud services, and APIs. The position is full-time and involves collaborating within an Agile environment to deliver high-quality software solutions.

Duties

  • Develop and maintain web applications using Laravel (PHP)
  • Build and integrate REST APIs and backend services
  • Work with JavaScript / TypeScript (Vue, React, or similar frameworks)
  • Develop and maintain Python-based services or scripts
  • Design and optimize databases using MySQL and PostgreSQL
  • Write clean, maintainable, and well-documented code
  • Debug, troubleshoot, and improve application performance
  • Collaborate with team members on features and architecture decisions

Qualifications

  • Proven knowledge/experience in web development, application development, and software development across multiple platforms.
  • Strong proficiency in PHP and Laravel, JavaScript, Python, Vue, React.
  • Hands-on experience with databases such as PgSQL, MySQL, or similar systems.
  • Familiarity with Linux cloud enviroments for deployment and management of applications.
  • Knowledge of RESTful API design and integration techniques.
  • Experience working within an Agile environment using tools like GitHub, or Git for version control.
  • Understanding of front-end technologies including HTML5, CSS3, and modern JavaScript frameworks.
  • Experience with Linux operating systems is preferred.
  • Ability to work collaboratively in a fast-paced environment while managing multiple priorities effectively.
  • This is a great opportunity for a motivated developer eager to work on real-world projects, collaborate with experienced developers, and strengthen your technical skills across the full stack.

Pay: $40,000.00-$45,000.00 per year

Benefits:

  • Casual dress
  • On-site parking
  • Paid time off

Work Location: In person
