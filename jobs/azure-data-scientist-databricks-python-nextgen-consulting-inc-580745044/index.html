About the Role

Build and deploy ML models and data pipelines on Azure Databricks.

About NextGen Consulting Inc.

Azure data analytics consultancy delivering cloud ML solutions.

Full Description

We are looking for Azure Data Scientist (Databricks & Python) for an enterprise scale client in Greater Toronto Area, Canada for an opportunity to work in state of the art, cutting edge technology environment.

Please apply with the resume clearly stating education and work experience including location of each assignment. Candidates must be authorized to work for any employer in Canada.

Key Responsibilities:

  • Develop and deploy machine learning models using Azure Machine Learning and Databricks.
  • Build data pipelines and perform ETL using Azure Data Factory and Databricks.
  • Conduct data preprocessing, feature engineering, and model tuning for large datasets.
  • Implement end-to-end ML workflows including training, validation, and deployment.
  • Collaborate with data engineers, analysts, and business teams to define requirements and deliver actionable insights.
  • Apply MLOps best practices for model versioning, monitoring, and CI/CD in Azure.
  • Create dashboards and visualizations using Power BI or similar tools.

Required Skills & Qualifications:

  • Strong proficiency in Python for data science and machine learning.
  • Hands-on experience with Azure Databricks, Azure Machine Learning, and Azure Data Lake.
  • Solid understanding of machine learning algorithms, deep learning, and statistical modeling.
  • Experience with Spark for distributed data processing.
  • Familiarity with MLOps, CI/CD pipelines, and Azure DevOps.
  • Excellent problem-solving and communication skills.

Preferred Qualifications:

  • Microsoft certifications such as Azure Data Scientist Associate or Azure Solutions Architect.
  • Experience with Power BI for visualization.
  • Knowledge of big data technologies and cloud security best practices
  • Bachelor’s or Master’s degree in Data Science, Computer Science, Statistics, or related field.
