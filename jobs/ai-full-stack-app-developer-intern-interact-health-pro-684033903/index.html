Full Description

AI FULL STACK APP DEVELOPER INTERN

(React Native, APIs & HealthTech Systems)

About Us

Supporting Recovery. Structuring Stronger Medical Cases

Interact Health PRO (IHP) coordinates care, connects clients with trusted providers, and supports the medical side of personal injury cases with clarity, compassion, and structure.

Our systems help organize:

  • Rehabilitation tracking
  • Medical documentation
  • Communication between clients, doctors, and legal teams
  • AI-supported workflows that improve recovery outcomes

We are actively building applications and digital systems that support real-world medical-legal workflows and virtual care coordination.

Role Overview

As an AI Full Stack App Developer Intern, you will help design, build, and enhance mobile and web applications that support personal injury recovery, rehabilitation tracking, and medical-legal coordination.

You will work on live projects including:

  • Personal Injury Recovery App
  • Concussion Management App
  • Weight Management Platform
  • Golf Performance & Recovery App

This role combines frontend and backend development with AI-powered workflows, automation systems, and user-focused experiences.

Key Responsibilities

Application Development

  • Build responsive mobile and web applications using React Native and modern frameworks
  • Develop backend systems, APIs, and integrations
  • Support secure and scalable application architecture
  • Troubleshoot bugs and optimize performance

AI-Enhanced Development

  • Integrate AI-assisted features into applications
  • Use Claude AI workflows to accelerate development and debugging
  • Support intelligent symptom tracking and automated workflows
  • Explore AI-assisted user experiences and recovery guidance tools

Systems Integration

  • Connect applications to databases and CRM systems
  • Support automation workflows using Airtable and Make.com
  • Ensure accurate synchronization of user and system data

Collaboration & Testing

  • Collaborate with UI/UX Designers, Analysts, and Project Managers
  • Participate in QA testing and feature validation
  • Contribute ideas for improving workflows and usability

AI Integration

You will actively use:

  • Claude AI Skills & Routines
  • Claude.md structured workflows
  • AI-assisted coding and debugging
  • Automation platforms including Airtable and Make.com

AI will be used to improve:

  • Development efficiency
  • Workflow automation
  • User engagement
  • Recovery tracking systems

Technical Skills

  • React Native / React.js
  • JavaScript / HTML / CSS
  • Node.js
  • APIs and integrations
  • MongoDB or SQL databases
  • GitHub version control

Assets

  • Mobile app deployment experience
  • Automation tools
  • Cloud platform exposure

Soft Skills & Professional Traits

  • Strong problem-solving mindset
  • Ability to work independently and collaboratively
  • Adaptability in a startup environment
  • Attention to detail and accountability
  • Strong written and verbal communication skills
  • Passion for technology and innovation

Learning Outcomes

  • Build real-world healthcare and medical-legal applications
  • Gain AI-integrated development experience
  • Learn automation and systems integration
  • Develop portfolio-ready projects
  • Collaborate with multidisciplinary teams

Why Join Us?

Become part of a 25+ IHP Intern Team and collaborate closely with Web Developers, Full Stack App Developers, AWS Engineers, Business Analysts, IT & Security Networking, Medical-Health, Social Media, Video Editors, and Administrative Groups. Together, we are launching Interact Health PRO’s innovative platforms to transform healthcare, patient outcomes, and virtual care technology.
