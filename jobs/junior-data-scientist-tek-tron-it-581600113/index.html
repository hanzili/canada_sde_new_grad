About the Role

Build and deploy data models, ETL pipelines, and ML solutions at scale.

About Tek Tron IT

Tek Tron IT delivers IT services and analytics solutions for diverse clients.

Full Description

Responsibilities

  • Collaborate with cross-functional teams to understand data needs and develop scalable data models
  • Design, implement, and optimize ETL (Extract, Transform, Load) processes to prepare large datasets for analysis
  • Develop machine learning models using frameworks like TensorFlow and Spark to solve complex problems such as natural language processing and unsupervised learning tasks
  • Conduct data mining and explore linked data sources to uncover hidden patterns and insights
  • Assist in model training, evaluation, deployment, and monitoring to ensure high performance in real-world applications
  • Work with cloud platforms like AWS for data storage, processing, and model deployment
  • Contribute to database design and management using SQL, SAS, or other database tools to support analytics initiatives
  • Use programming languages such as Python, R, Java, C, Bash (Unix shell), or VBA for scripting and automation tasks
  • Support big data projects utilizing Hadoop or Spark frameworks for efficient processing of massive datasets
  • Create visualizations and dashboards with tools like Looker to communicate findings clearly to stakeholders

Experience

  • Proven experience or academic background in data science, machine learning, or related fields with a focus on AI applications
  • Hands-on knowledge of machine learning frameworks such as TensorFlow or similar tools for model development
  • Familiarity with cloud services like AWS for data storage and model deployment is highly desirable
  • Strong understanding of SQL for database querying along with experience in ETL processes and database design principles
  • Experience working with big data technologies including Hadoop or Spark is a plus
  • Knowledge of natural language processing techniques and unsupervised learning methods enhances your fit for this role
  • Programming proficiency in Python and R; experience with Java or C is advantageous for integrating systems
  • Ability to analyze large datasets using data mining techniques and advanced analytics methods
