Frontend Engineer Intern

at Arkion Identity Systems

About Arkion Identity Systems

Canada-based cybersecurity platform startup

Full Description

Frontend Engineer Intern

Internship Type:  Paid Internship

About The Role

This is a paid internship. We are building a secure, enterprise-grade web platform for managing complex operational workflows, access controls, automation, reporting, and system monitoring. As a Frontend Engineer Intern, you will help create clean, reliable, and intuitive interfaces for technical and administrative users.

What You’ll Work On

  • Build frontend pages for dashboards, resource management, settings, reports, automation, and monitoring.
  • Create reusable UI components such as tables, forms, modals, filters, search, sorting, sidebars, and bulk action menus.
  • Integrate frontend screens with backend APIs.
  • Improve permission-aware navigation and role-based user experiences.
  • Help test key workflows such as login, onboarding, approvals, downloads, settings, and admin actions.
  • Fix UI bugs, polish interactions, and improve usability across complex workflows.
  • Collaborate with backend engineers, designers, and product stakeholders.

What We’re Looking For

  • Hands-on experience with TypeScript, React, and Next.js.
  • Strong understanding of HTML, CSS, responsive design, and component-based UI development.
  • Ability to work with REST APIs and JSON data.
  • Good attention to detail when working on security-sensitive workflows.
  • Clear communication and willingness to learn in a fast-moving technical environment.
  • Must currently be based in Canada.
  • Must be legally authorized to work in Canada without visa sponsorship.

What You’ll Learn

  • You’ll gain hands-on exposure to real-world cybersecurity workflows and production software practices that complement and go beyond classroom learning.
  • How secure, enterprise-grade platforms are built.
  • How frontend apps integrate with complex backend systems.
  • How permissions, roles, and sensitive workflows shape UI design.
  • How to build reliable admin tools for technical users.

Nice To Have

  • Experience building dashboards, admin portals, or data-heavy interfaces.
  • Familiarity with authentication, permissions, role-based access, or multi-tenant applications.
  • Basic knowledge of Docker, Git, or API documentation.
  • Interest in UX for technical users and enterprise software.

Location & Work Authorization

  • Candidates must currently reside in Canada.
  • We do not provide visa sponsorship.

Confidentiality Note

This role involves confidential product and technical information. Candidates may be asked to sign an NDA. Internal project details, architecture, source code, customer information, credentials, and documentation must be treated as confidential.
