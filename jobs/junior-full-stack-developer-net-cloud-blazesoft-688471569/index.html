About Blazesoft

Full Description

Job Description

Join Blazesoft, a fast-growing and innovative company building cutting-edge gaming platforms and experiences. We are passionate about technology, scalability, and creating high quality products that reach thousands of users worldwide. Our culture is collaborative, fast-moving, and engineering-driven, with plenty of opportunities to learn and grow.

Position Overview

We are looking for a motivated Junior Full Stack Developer to join our development team. In this role, you will work on modern backend and frontend systems using the Microsoft stack, contribute to cloud-native applications running on AKS (Azure Kubernetes Service), and collaborate with experienced engineers across multiple domains.

This is a great opportunity for a developer early in their career who wants real hands-on experience with scalable systems, cloud infrastructure, and modern development practices.

Key Responsibilities

  • Develop and maintain backend services using C# and .NET
  • Design, develop, and maintain backend services supporting third-party integrations with gaming operators and providers
  • Build and enhance frontend applications using modern web technologies
  • Work with cloud-native applications deployed on Azure and AKS
  • Take part in the full development lifecycle, from design to production
  • Work closely with developers and stakeholders to deliver new features quickly and efficiently
  • Troubleshoot issues across the application stack and production environments
  • Write clean, maintainable, and testable code
  • Participate in technical discussions, code reviews, and continuous improvement initiatives Contribute to a fast-moving engineering environment with real ownership and impact

Requirements

  • 1+ years of software development experience, internships, or relevant personal projects
  • Knowledge of C# and .NET / ASP.NET Core
  • Familiarity with frontend development concepts (React, Angular, or similar)
  • Basic understanding of REST APIs and microservices architecture
  • Familiarity with SQL and NoSQL databases
  • Understanding of Git and modern development workflows
  • Eagerness to learn cloud technologies and DevOps practices Strong problem-solving skills and a positive attitude

Nice to Have

  • Experience with Azure or AWS
  • Familiarity with Docker and Kubernetes
  • Experience working with CI/CD pipelines Knowledge of Redis, messaging systems, or distributed systems concepts

Benefits

  • Work with modern technologies and cloud-native systems
  • Learn from experienced engineers and architects
  • Friendly and collaborative engineering culture
  • Opportunity to grow quickly and make real impact

  • Flexible and dynamic work environment
