About Helic & Co.

Emerging software team with mentorship-focused culture

Full Description

This is a remote position.

Overview

We are seeking a motivated Junior Software Engineer to contribute to software development projects, focusing on coding, testing, and maintenance. This entry-level role offers mentorship to build technical skills in a collaborative team environment.

Key Responsibilities

  • Write clean, efficient code based on specifications using languages like Python, Java, or JavaScript.
  • Debug, test, and optimize software applications and features.
  • Collaborate with senior engineers on design, code reviews, and deployments.
  • Maintain codebases, update documentation, and support production issues.
  • Participate in agile processes, including sprints and stand-ups.

Qualifications

  • Bachelor’s degree in Computer Science, Software Engineering, or equivalent.
  • Basic proficiency in programming, Git, and SDLC; projects or internships preferred.
  • Strong problem-solving and eagerness to learn.
  • Good communication skills.

Preferred Attributes

  • Experience with frameworks (e.g., React, Spring) or cloud services (e.g., AWS).
  • Contributions to open-source or personal coding projects.

What You’ll Gain

  • Practical development experience on real products.
  • Mentorship for rapid growth to mid-level engineering roles.
