Full Description

Job Title: Junior System Software Engineer Job Type: Onsite Job Location: Vancouver, British Columbia, Canada Experience: 1 to 3 Years Pay Range: CA $35-$40 PH Role Summary: NearSource is seeking a *Junior* System Software Engineer**** to support enterprise-grade CI microservice systems and cloud-based infrastructure. The selected candidate will contribute to platform stability, feature engineering, and AI-driven components within a structured on-site engineering environment.

Key Responsibilities:

  • Operate and maintain CI microservice systems in a production setting.
  • Monitor system performance, troubleshoot incidents, and resolve technical issues.
  • Execute regular security patching for servers and databases.
  • Architect and implement enhancements to existing platform capabilities.
  • Diagnose and remediate system defects to ensure reliability and uptime.
  • Support CI/CD pipelines and collaborate with cross-functional engineering teams.

Must-Have Skills

  • Bachelor's degree in Computer Science or a related discipline.
  • 1 to 3 years of experience in software engineering or system operations.
  • Proficiency in Linux environments.
  • Develop new features, including AI-related components such as MCP servers and AI agents.
  • Working experience with scripting languages, Python or Javascript.
  • Hands-on experience with Docker.
  • Foundational understanding of databases and CI/CD methodologies.
  • Exposure to Azure or AWS cloud platforms.

Nice-to-Have Skills:

  • Experience supporting microservices in production environments.
  • Familiarity with automation workflows and infrastructure processes.
  • Exposure to AI components or agent-based architectures.
  • Understanding of cloud monitoring and observability tools.

Certifications

  • Microsoft Azure Fundamentals, AWS Cloud Practitioner, or related cloud certifications.
