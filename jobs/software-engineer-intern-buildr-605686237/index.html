Full Description

Software Engineer Intern — buildr

Location: Remote (Canada)

Type: Internship (3–6 months)

About buildr

buildr is an early-stage startup building a software platform to improve operational efficiency for Canadian construction professionals. We’re currently in stealth mode and looking for an intern to help build core product infrastructure.

What you’ll do

You’ll work directly with the founding team to develop key features and backend systems for a SaaS product. You will:

  • Build and maintain backend services and APIs
  • Develop user interfaces and internal tools
  • Design and implement data models and workflows
  • Integrate with external services and APIs
  • Write tests, documentation, and maintain clean code
  • Help improve deployment, monitoring, and reliability

What we’re looking for

  • Currently studying Computer Science, Software Engineering, or a related field
  • Strong fundamentals in at least one programming language (Python, JavaScript/TypeScript, or similar)
  • Familiarity with web frameworks (e.g., Node.js, Django/Flask, React, Next.js)
  • Comfortable with REST APIs and databases (PostgreSQL, MongoDB, etc.)
  • Curious, proactive, and able to work independently
  • Strong communication skills

Nice-to-haves

  • Experience with workflow automation or workflow engines
  • Document generation (PDF, forms)
  • Cloud services (AWS, GCP, Azure)
  • Docker / Kubernetes
  • Interest in construction, real estate, or regulation

What you’ll gain

  • Real-world experience building a product from the ground up
  • Ownership of meaningful features and infrastructure
  • Mentorship from experienced engineers and founders
  • A strong portfolio for future opportunities

How to apply

  • Send a short note explaining why you’re interested to our company profile.
