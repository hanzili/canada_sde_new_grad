About the Role

Develop Android apps, integrate with hardware SDKs

About 44degNorth Technologies Incorporated

Industrial IoT hardware-focused company in Oakville

Full Description

We’re looking for a hands-on Android developer who wants more than just UI tweaks and textbook projects. At 44North, you’ll work on live Android apps that interact with real-world hardware—from RFID scanners to IoT devices used in manufacturing, oil & gas, and asset tracking environments.

If you’ve ever wanted to build apps that talk to actual hardware, not just simulate it, this is your chance.

What You’ll Work On:

  • Build and refine Android apps used daily by field teams in industrial environments.

  • Integrate with barcode/RFID scanners and IoT hardware using manufacturer SDKs.

  • Handle structured data: read from Excel/CSV files, transfer it through API calls.

  • Collaborate with designers via Figma to implement pixel-perfect UIs.

  • Push code regularly to GitHub, and keep tasks updated in Jira.

What You Bring:

  • You’ve built Android apps using Java or Kotlin (show us your GitHub or project links).

  • You understand REST APIs and have handled data from CSV or Excel sources.

  • You’re familiar with GitHub and Jira, and you document your work properly.

  • You’re curious about hardware SDKs and like figuring out how software connects to devices.

  • You’re here to build real-world experience that leads to more than a line on your resume.

Bonus if you’ve already worked with SDKs to connect Android apps to external devices.

Final-year students and recent grads are welcome.

Job Type: Internship / Co-op
Contract length: 2-3 months

Pay: $18.00-$20.00 per hour

Benefits:

  • Company events

Work Location: In person
