  • Build and maintain Copilot Studio agents that support Engineering and Operations workflows (Q&A, guided work, task routing, standard responses).
  • Design agent topics and conversation flows including trigger phrases, clarifying questions, error handling, and escalation paths.
  • Connect agents to approved business systems using actions and connectors (examples: SharePoint Lists, Teams, Outlook, Dataverse).

  • Develop test plans and execute iterative testing (happy paths, edge cases, access checks, regression testing) and track improvements.
  • Monitor agent performance and adoption (usage, success rate, top intents, failure reasons) and implement continuous improvement.
  • Document agent design and support materials (architecture, knowledge sources, actions, permissions, release notes, user guide).
  • Automate current business systems and workflows to reduce manual effort and improve data accuracy (Power Automate and basic scripting as applicable).
