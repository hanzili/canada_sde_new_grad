Job Overview
We are seeking a motivated Junior Software Engineer to join our dynamic development team. This entry-level position offers an excellent opportunity to gain hands-on experience in software development, web applications, and cloud technologies. The ideal candidate will have a passion for coding, problem-solving skills, and a desire to grow within a collaborative environment. You will contribute to the design, development, and maintenance of innovative software solutions across various platforms.

Duties

  • Assist in developing and maintaining web applications using JavaScript, Ruby on Rails, and other front-end technologies.
  • Support application development projects utilizing languages such as Java, C#, Python, C++, and VBA.
  • Collaborate with team members to implement RESTful APIs and integrate third-party services via APIs.
  • Write efficient SQL queries and work with databases like MySQL and SQL Server to manage data storage solutions.
  • Participate in Agile development processes, including sprint planning, stand-ups, and code reviews.
  • Use version control systems such as GitHub, SVN, and Git to manage code repositories effectively.
  • Deploy applications on cloud platforms including AWS and Azure while ensuring scalability and security.
  • Contribute to Linux-based environments for application deployment and testing.
  • Assist in troubleshooting issues, performing code reviews, and documenting technical specifications.
  • Engage in continuous learning of new tools, frameworks, and best practices to enhance development skills.
