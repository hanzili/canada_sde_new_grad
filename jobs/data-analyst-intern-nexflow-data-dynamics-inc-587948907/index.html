About NexFlow Data Dynamics Inc.

Analytics-focused, data-driven company with diverse clients

Full Description

Overview
We are seeking a detail-oriented and analytical Data Analyst to join our team. The ideal candidate will be responsible for transforming data into actionable insights to support strategic decision-making across various departments. This role requires a strong foundation in data analysis, database management, and visualization tools, with an emphasis on delivering high-quality reports and dashboards. The position offers an opportunity to work on diverse projects utilizing advanced analytics and data management techniques in a dynamic environment.

Responsibilities

  • Collect, clean, and analyze large datasets using SQL, R, Python, and other tools to identify trends and patterns.
  • Develop and maintain interactive dashboards and reports using Tableau, Power BI, and other visualization platforms.
  • Collaborate with business stakeholders to understand requirements and translate them into technical specifications.
  • Design and optimize databases, including Microsoft SQL Server and Oracle, ensuring data integrity and efficiency.
  • Conduct business analysis to identify opportunities for process improvements and data-driven solutions.
  • Manage projects within Agile frameworks, adhering to SDLC best practices for software development lifecycle management.
  • Create detailed documentation of data models, workflows, and analysis procedures using Visio or similar tools.
  • Support project management activities by tracking progress, managing timelines, and coordinating cross-functional teams.
  • Utilize VBA and Bash (Unix shell) scripting for automation of repetitive tasks and data processing workflows.

Skills

  • Proficiency in data visualization tools such as Tableau and Power BI.
  • Strong knowledge of SQL, including Microsoft SQL Server and Oracle databases.
  • Experience with programming languages like R and Python for statistical analysis and modeling.
  • Familiarity with SDLC processes, Agile methodologies, and project management principles.
  • Business analysis expertise to interpret data within the context of organizational goals.
  • Database design skills to develop efficient schemas that support analytical needs.
  • Data analysis skills with a focus on deriving insights from complex datasets.
  • Knowledge of VBA scripting for automation purposes.
  • Experience working with Unix shell scripting (Bash) for data processing tasks.
  • Excellent analysis skills combined with attention to detail to ensure accuracy in reporting. This position offers a challenging environment where innovative thinking is valued, providing opportunities for professional growth in the field of data analytics while contributing significantly to organizational success.
