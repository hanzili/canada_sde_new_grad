Full Description

Job Title: Junior Software Developer

Company: Naryant

Location: Oakville, ON

Type: Full-time; Hybrid

About Naryant

Naryant is a digital transformation partner, leading organizations to reimagine, build, and evolve their strategic operations using data. Naryant serves a broad spectrum of clients in healthcare, fleet management, transportation, municipal energy management, and more. Our agile design approach and focus on creating and integrating multiplatform software, web, mobile, machine learning and IoT solutions allows us to accelerate cross-enterprise decision-making, delivering solutions that define the future of technology enablement.

Naryant’s Purpose Statement:

Data for a safer, healthier, and more sustainable community.

Role Overview:

The Junior Software Developer will be pivotal in writing, modifying, reviewing, optimizing, and debugging applications developed with the MVC framework, CSS, jQuery, and Bootstrap. Your contributions will span all tiers of application architecture, with a focus on creating scalable solutions that drive data enablement across diverse application fields.

Key Responsibilities:

  • Collaborate with our development team on a system redesign of a best-in-class, large-scale data collection and analysis platform.

  • Improve and streamline current architecture, including class structures, business workflows, and information security.

  • Contribute to the accelerated adoption of machine learning and enhance interface design.

  • Work on enhancing and optimizing ASP .NET MVC framework applications, using technologies like C#, CSS, jQuery, and SQL Server.

Required Skills:

Experience with:

C# and .NET (Core / Framework / ASP.NET MVC / Web API / Entity Framework) Microsoft SQL Server

  • JavaScript / TypeScript

  • HTML5/CSS

  • Experience using Git and contributing to CI/CD pipelines

  • Solid understanding of software design patterns and best practices.

Strong communication skills with the ability to adapt to different audiences. Ability to concisely present and summarize experimental results

Nice-to-Have Qualifications:

Experience working with:

  • React.js

  • Reporting technologies (SSRS / PowerBI)

  • Cloud platforms (Microsoft Azure / AWS)

  • Experience with Linux (Ubuntu)

  • Experience with DevOps or infrastructure automation

  • A strong aptitude for math and statistics.

Why Naryant?

  • Work on the cutting edge of data analytics and IoT-driven solutions for industries such as fleet management and healthcare.

  • Collaborative and innovative work environment with a focus on empowering clients through technology.

  • Competitive compensation and benefits package.
