Job Description Change the world. Love your job. About The Job Our Front-end Development internship will provide you with a variety of experiences tailored to the latest trends in front end technologies such as HTML, CSS and JavaScript frameworks. You'll learn to translate mock-ups into functional components while mastering development workflows, version control, and cross-browser compatibility. This is a unique opportunity to gain hands-on experience implementing design systems, writing maintainable code, and optimizing performance for actual customers who regularly access the website of an $18B technology company. This internship might be for you if you have an interest in:

  • Developing and driving a best-in-class website for customers looking for information to evaluate products, make purchases and get timely support
  • Developing front-end strategies to support our UX Designers in providing consistent web design experiences to help promote the TI brand
  • Working with IT to launch new innovative, technically credible solutions on TI.com
  • Analyzing user data to evaluate front-end component effectiveness and collaborating with senior developers to optimize web performance

Qualifications Minimum Requirements:

  • Pursuing a Bachelors degree in Web Design & Development, User Experience Design, Computer Engineering or Computer Science with focus in marketing, or related degree.
  • Minimum GPA 3.0/4.0 GPA

Preferred Qualifications

  • Experience with HTML, CSS, JavaScript semantics and modern web framework such as Angular, Stencil or REACT, connecting with APIs
  • Strong online portfolio demonstrating a range of web-related development work a must
  • Experienced with C, Java, Node.js, npm, Git and DevOps processes
  • Understands fundamentals of user experience design principles, UX research, web development and web production
  • Comfortable with UNIX command line interface
  • Understanding of responsive design, cross-browser compatibility, and web accessibility
