About the Role

Build and maintain cloud infra and data pipelines

About TELUS Digital

Canadian telecom, digital customer experience arm

Full Description

Requirements Description and Requirements Requirements:

  • Proficient in IAC especially Terraform focusing on GCP Data services and GCP networking (VPC, PSC, PSA, NCC, IPSEC, Security)
  • Proficient in Kubernetes building and deployment through Helm Charts.
  • Proficient with Application Pipeline building and deploying CI/CD through Github Action and Cloud build.
  • Having experience to embed Application Pipeline code security.
  • Must be capable of troubleshooting issues( Logging, open-source clues, reviewing knowledge base).
  • Must be Hands-on-experience on GCP terraform provider and products with enterprise customers or projects.
  • Good Knowledge of Cloud/GCP security principles

Data Engineering:

  • Must be able to know and have experience on key GCP Data services (Bigquery, DataFlow, DataStream, Pub/Sub, workflow, EventArch, etc).
  • Must be able to review and implement data pipeline code. (Python, nodejs, java)
  • Must be able to setup Observability Pipeline through IAC Github and Github Actions
  • Must be able to understand traditional Databases especially Oracle and Mysql.
  • Must be able to write SQL stored procedure
  • Must be having knowledge of CRM -Siebel

Interpersonal Skills:

  • Must be flexible for different Time Zone especially EST and MST.
  • Must be able to articulate the blocker, challenges for the tasks or solution
  • Having the capacity to work independently in a stipulated time period.

EEO Statement

At TELUS Digital, we enable customer experience innovation through spirited teamwork, agile thinking, and a caring culture that puts customers first. TELUS Digital is the global arm of TELUS Corporation, one of the largest telecommunications service providers in Canada. We deliver contact center and business process outsourcing (BPO) solutions to some of the world's largest corporations in the consumer electronics, finance, telecommunications and utilities sectors. With global call center delivery capabilities, our multi-shore, multi-language programs offer safe, secure infrastructure, value-based pricing, skills-based resources and exceptional customer service - all backed by TELUS, our multi-billion dollar telecommunications parent.
