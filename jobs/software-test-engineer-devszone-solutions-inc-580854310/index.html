About the Role

Hands-on software testing with automation, QA, and CI/CD support.

About DevsZone Solutions Inc.

DevsZone Solutions Inc., Montreal-based software quality and QA services provider.

Full Description

Job Overview
We are seeking a highly skilled and detail-oriented Software Test Engineer to join our dynamic development team. The ideal candidate will be responsible for designing, executing, and maintaining comprehensive test plans to ensure the quality and functionality of our software products across various platforms, including mobile and desktop applications. This role offers an exciting opportunity to work with cutting-edge technologies and contribute to the delivery of reliable, high-performance software solutions.

Duties

  • Develop and implement automated and manual test cases using tools such as TestNG, Selenium, JUnit, and Appium for mobile applications on Android and iOS platforms.
  • Perform functional, regression, integration, and performance testing across diverse operating systems including Windows, Linux, and cloud environments like AWS and Azure.
  • Collaborate with software developers to identify bugs, troubleshoot issues, and verify fixes through tools like Visual Studio, Eclipse, and IntelliJ IDEA.
  • Utilize version control systems such as Git, GitLab, GitHub, SVN, TFS, and Jenkins for continuous integration and deployment pipelines aligned with DevOps practices.
  • Conduct database testing using MySQL, SQL Server, Oracle, PL/SQL, and JDBC to validate data integrity and backend processes.
  • Analyze system requirements to develop comprehensive test strategies that cover REST APIs, UI/UX components (HTML, CSS, JavaScript), and backend services.
  • Maintain detailed documentation of test cases, defect reports via Jira or TFS, and test results for review by stakeholders.
  • Participate in code reviews and provide feedback to improve overall software quality.
  • Support the automation of testing frameworks using Node.js, C#, C++, Python, Bash scripts (Unix shell), Maven, and other relevant tools.
  • Stay updated on industry best practices in software testing methodologies and emerging technologies such as TDD/BDD frameworks.

Experience

  • Proven experience in software testing within a fast-paced development environment.
  • Hands-on expertise with automation tools like Selenium WebDriver, TestNG, JUnit, Appium for mobile testing on Android and iOS devices.
  • Strong programming skills in JavaScript, Java, Python, C#, C++, PHP or related languages used in test automation scripts.
  • Familiarity with database management systems including MySQL, SQL Server (Microsoft SQL Server), Oracle (PL/SQL), and related query languages.
  • Experience working with cloud platforms such as AWS or Azure for testing distributed applications.
  • Knowledge of DevOps tools including Jenkins or GitLab CI/CD pipelines for automated build/test deployment workflows.
  • Understanding of computer networking concepts relevant to application performance testing.
  • Excellent analysis skills with the ability to troubleshoot complex issues across multiple environments.
  • Experience with UI/UX testing involving HTML5/CSS3/JavaScript frameworks like Node.js or MVC architecture is a plus. Join our team to contribute your expertise in software quality assurance while working on innovative projects that impact millions of users worldwide!
