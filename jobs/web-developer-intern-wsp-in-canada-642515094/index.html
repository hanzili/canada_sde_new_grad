The Opportunity Build the Future With Us. As a Web Developer Co‑op Student supporting our Building Structures team , you will help design and develop a digital tool that improves how our engineers collaborate, model, and deliver projects across Canada. This is a chance to make a real‑world impact by creating solutions used by technical experts shaping skylines and infrastructure every day.

Your Impact

  • Contribute to designing and developing a web‑based tool that supports building structure workflows.
  • Collaborate with structural engineers to understand user needs and translate them into digital functionality.
  • Build efficient, user‑friendly front‑end interfaces that simplify complex engineering tasks.
  • Assist in back‑end development, API connections, and data handling to enable seamless tool performance.
  • Test, debug, and enhance prototypes throughout the co‑op term.
  • Document features, workflows, and technical considerations for future iterations.
  • Apply creativity and innovation to streamline engineering processes and improve project delivery.

The Skills That Set You Apart

  • Currently enrolled in a university program in Computer Science, Software Engineering, Computing Science, or a related field.
  • Experience with JavaScript, Typescript, Next.JS, HTML, CSS, and at least one front‑end framework (e.g., React, Vue, Angular).
  • Knowledge of back‑end technologies (Node.js, Python, or similar) is an asset.
  • Comfortable working with APIs and version control (Git).
  • Competency in creating and managing SQL databases.
  • Curious, collaborative, and eager to learn from technical experts in a real engineering environment.
