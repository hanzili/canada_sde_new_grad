About NetNow Inc.

Early-stage startup; fintech B2B payments platform

Full Description

The Role

NetNow is seeking a Full Stack Developer

to join our fast-growing team. We’re looking for mission-driven engineers who want to work end-to-end, building intuitive user experiences while helping scale the infrastructure that powers modern B2B credit and payments.

As a junior engineer, you’ll work closely with senior engineers, product managers, and designers to ship high-impact features. You’ll contribute across the stack — from frontend to backend to infrastructure — while developing a strong foundation in building scalable production systems.

This is a high-ownership role in an early-stage, fast-growing startup environment where your work will directly impact customers.

What You’ll Do

  • Design, build, test, and deploy new features used by credit and finance teams across North America
  • Contribute to both frontend and backend systems across our platform
  • Support infrastructure improvements that help us scale reliably and securely
  • Write clean, maintainable, and well-tested code
  • Participate in code reviews and technical design discussions
  • Collaborate cross-functionally with product, operations, and customer teams
  • Learn and grow quickly in a fast-paced startup environment

What We’re Looking For

  • Bachelor’s degree in Computer Science, Software Engineering, or a related field (if new grad, latest graduation year must be 2026)
  • Internship, co-op, or prior work experience building software in a professional setting
  • Experience building and shipping web applications (through internships, research, or substantial side projects)
  • Strong computer science fundamentals (data structures, algorithms, system design basics)
  • Experience with Python and/or JavaScript
  • Familiarity with modern frontend frameworks (e.g., React)
  • Exposure to cloud platforms (e.g., GCP, AWS, or Azure) is a plus
  • Familiarity with relational databases (e.g., Postgres)
  • Strong communication skills and ability to work in a remote-first, collaborative environment
  • Eagerness to learn, take ownership, and grow into increasing responsibility

Our Tech Stack

  • Python
  • JavaScript / React
  • GCP
  • Postgres

We care more about strong fundamentals and learning ability than exact stack experience.
