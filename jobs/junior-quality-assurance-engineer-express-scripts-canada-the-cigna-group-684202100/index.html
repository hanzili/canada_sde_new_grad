Job Summary

Express Scripts Canada is actively looking to hire a Junior Quality Assurance Engineer. This role will work in a team to help maintain quality controls in various Express Script Canada’s applications for the Private BU. The Junior QA Engineer will work in a team comprising of Business Analyst, Developers under the supervision of a senior member to help test applications. The primary responsibility of the Junior QA Engineer is to write test cases, create automation scripts and perform manual testing wherever applicable. The Junior QA Engineer will also be contributing towards maintenance and improvement of the Express Scripts Canada’s automation test suite.

Key Responsibilities:

  • Execute manual and automated test cases based on defined test plans
  • Support functional, regression, and integration testing activities
  • Assist in developing and maintaining automation scripts under guidance
  • Identify, log, and track defects with clear documentation
  • Collaborate with developers and BAs to understand requirements and validate features
  • Maintain test cases, test data, and provide regular status updates to the team

What We’re Looking For:

  • A postsecondary diploma/degree in Computer Science, Computer Programming, or a related field
  • Minimum of 1 year of quality assurance experience (professional or internship/co-op); considered a strong asset
  • Strong knowledge of SQL/ PL-SQL and relational databases (e.g., Oracle
  • Hands-on experience in automation testing using Selenium or Playwright
  • Hands-on experience in REST API testing using Postman or Bruno
  • Basic understanding of Core Java (Java 17 or later) and the Spring Boot framework
  • Basic understanding of front-end technologies such as HTML5, CSS, and JavaScript
  • Familiarity with Git (version control) and basic CI/CD concepts using DevOps tools
  • Ability to learn new skills and adapt to rapidly changing environment
