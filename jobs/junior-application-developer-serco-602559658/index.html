Position Description & Qualifications

Purpose of the Job:

We are seeking a motivated developer to support design, development, and deployment of innovative software solutions, with a strong emphasis on AI-driven tools and applications. This role is ideal for a curious and growth-oriented individual eager to expand their expertise in modern development environments. The successful candidate will work with cloud platforms, AI/ML technologies, and full-stack frameworks, gaining hands-on experience with cutting-edge AI capabilities while contributing to mission-critical business initiatives.

Primary Duties and Responsibilities

  • Assist in developing, testing, and deploying applications using Python, React, and Tailwind CSS.
  • Work with Azure AI tools and services, contributing to AI model integration and automation solutions.
  • Contribute to backend development using .NET technologies such as C#, ASP.NET Core, and Entity Framework, supporting scalable enterprise solutions.
  • Support and maintain non-relational databases (e.g., MongoDB, Azure Cosmos DB) for flexible data storage and retrieval.
  • Leverage Azure services including Azure Functions, Logic Apps, and Azure DevOps to enhance cloud-native application development and deployment.
  • Support the design and maintenance of relational databases (SQL Server and Oracle).
  • Create and maintain reports and dashboards in Power BI, learning data visualization best practices.
  • Participate in the development and experimentation of machine learning models.
  • Work closely with senior developers to troubleshoot, debug, and optimize solutions.
  • Collaborate in Agile/Scrum teams, contributing to sprint planning, stand-ups, and code reviews.

  • Familiar with various authentication providers such as Firebase Authentication, Auth0, and Azure Active Directory for secure user access and identity management.

  • A bachelor’s degree in information technology / computer science or related disciplines e.g. Engineering

  • 1 -2 years of experience are preferred.
  • Strong foundational knowledge of Python and modern JavaScript frameworks (React).
  • Experience with Tailwind CSS for responsive UI design.
  • Familiarity with SQL Server and Oracle relational databases.
  • Exposure to Azure services; understanding of Azure AI tools is an advantage.
  • Knowledge of .NET framework considered a strong asset.
  • Willingness to learn Power BI for reporting and visualization.
  • Basic understanding of machine learning concepts and enthusiasm to learn AI/ML development.
  • Strong problem-solving, communication, and teamwork skills.
  • Passion for continuous learning and growth in modern development technologies.
