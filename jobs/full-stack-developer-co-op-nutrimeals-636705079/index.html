Full Description

Full Stack Developer Co-op

Company: Nutrimeals Inc. (DBA Nurish'd)

Location: Calgary (In office M-T, WFH Friday)

Duration: 8 months, with opportunity to continue full time after the project.

About the 1st project:

Nurish'd is a health-tech platform connecting users with personalized meal plans, healthy meals vendors, and registered dietitians. We've built an in-house Electronic Health Record (EHR) system to support our dietitians with structured client data, consultation workflows, and secure health documentation. We are looking for a full stack developer that will be frontend-focused to re-build and deploy core components of our internal EHR platform from a completed Figma design.

Role Overview

This role is primarily frontend-focused (70%) with exposure to full stack development. You will contribute directly to the development

● Manage client health profiles

● Document consultations

● Track nutrition plans and progress

● Review structured medical and lifestyle data

● Integrate AI-assisted insights safely

● & more

This is a high-responsibility role involving sensitive health data workflows. Attention to detail and clean architecture are critical.

Responsibilities

● Develop secure, responsive UI components using Next.js and TypeScript

● Build structured data entry interfaces (forms, validation, dynamic sections)

● Implement complex state-driven workflows (multi-step health records, consultation notes)

● Integrate frontend components with FastAPI backend services

● Ensure clean separation between PHI-related data and general application state

● Collaborate with backend engineers to define API contracts

● Improve UX for clinical workflows (efficiency, clarity, usability)

● Participate in code reviews and follow structured Git workflows

Required Qualifications

● Strong proficiency in TypeScript, React, and Next.js

● Experience building structured forms and data-heavy interfaces

● Solid understanding of component architecture and state management

● Experience consuming REST APIs

● Good understanding of frontend validation and error handling

● Strong attention to detail

● Currently enrolled in Computer Science, Software Engineering, or related field

Nice to Have

● Experience with Python or FastAPI

● Familiarity with MongoDB

● Exposure to healthcare, HIPAA, PHI, or secure systems

● Familiarity with authentication flows (JWT, RBAC)

● Experience with cloud platforms (Azure preferred)

What You’ll Work On

● Dietitian dashboards

● Structured consultation documentation interfaces

● Client health profile management

● Secure role-based access views

What We’re Looking For Someone who:

● Writes careful, structured code

● Uses AI tools to expedite the development, but practises mindful code review before deploying

● Curious about building production health-tech systems

Job Types: Full-time, Permanent

Pay: $58,000.00-$70,000.00 per year

Benefits:

  • Casual dress
  • Discounted or free food
  • Paid time off
  • Store discount

Ability to commute/relocate:

  • Calgary, AB T2H 2J6: reliably commute or plan to relocate before starting work (required)
