About the Role

Configure Linux AI server, deploy internal apps

About Curex Medical Centre

Healthcare clinic building internal AI infrastructure

Full Description

Job Summary
We are a high-volume medical clinic in Kitchener looking for a technically curious and self-directed co-op student to assist with several internal software, automation, and AI infrastructure projects.

This role is ideal for an undergraduate student in Computer Science, Software Engineering, Systems Design Engineering, or a related field who enjoys building practical tools and working on real-world infrastructure problems.

The position will begin with 1–2 days onsite for setup and planning, with the remainder primarily remote and flexible.

You will work directly with a physician who is actively building internal automation and AI systems for clinical workflows.

Responsibilities

  • Configure and maintain a Linux-based local AI server
  • Assist with secure remote access setup and networking
  • Work with locally hosted LLMs
  • Help deploy internal applications accessible throughout the clinic
  • Optimize reliability and usability for non-technical staff

EMR Automation / Refill Processing System

Assist with development of a browser-based prescription refill processing platform that interfaces with the OSCAR EMR through browser automation.

Potential technologies include:

  • Playwright or Selenium
  • OCR/document processing
  • Python backend development
  • HTML/CSS/JavaScript frontend work
  • Local AI model integration

This project is highly exploratory and involves significant architecture and workflow design.

Additional Projects

  • Deployment of internal Python tools
  • Browser-based clinic utilities
  • Refactoring and managing a large Tampermonkey automation ecosystem
  • Workflow automation projects
  • Internal tooling and dashboard development
  • Additional AI-assisted clinic workflow projects

We are looking for someone who is:

  • Comfortable working independently
  • Interested in AI tooling and practical automation
  • Curious about systems/infrastructure work
  • Comfortable troubleshooting unfamiliar technologies
  • Interested in building tools that solve real operational problems in a health care setting

Strong candidates may have experience with some of the following:

  • Python
  • JavaScript/TypeScript
  • Linux/Ubuntu
  • HTML/CSS
  • Browser automation (Playwright/Selenium)
  • Docker
  • Local LLMs / AI tooling
  • Networking or server administration

You do not need experience in all areas.

Position Details

  • Hybrid position (initial onsite work required in Kitchener)
  • Flexible scheduling
  • Primarily project-based work
  • Opportunity for ongoing part-time work after co-op term
  • Significant autonomy and ability to shape projects
  • This is an excellent opportunity to gain direct, practical experience implementing novel technology in a real-world healthcare environment — an ecosystem that is traditionally difficult to break into from the technology side.
  • Claude Code or Codex max subscription provided

Please include:

  • Resume
  • GitHub or project portfolio (if available)
  • Brief description of your technical interests/projects

Pay: $22.00-$25.00 per hour

Benefits:

  • Flexible schedule

Work Location: Hybrid remote in Kitchener, ON N2G 1E2
