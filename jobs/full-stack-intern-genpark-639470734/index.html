About GenPark

GenPark, AI-driven brand discovery platform.

Full Description

Company Description

GenPark is an innovative agent-driven discovery platform designed for global commerce. By leveraging Intelligent Brand Agents and User Agents, GenPark continuously learns from behavior and context to connect consumers with the most relevant products, content, and experiences. Using personalized AI content, interactive agents, and immersive videos, the platform empowers brands to better engage with the Gen Z audience. GenPark enhances efficiency and fosters meaningful engagement for modern marketing strategies.

Role Description

This is a hybrid internship role for a Full Stack Intern, located in Toronto, ON, with opportunities for partial remote work. The intern will assist in developing, testing, and deploying scalable web applications, collaborate with cross-functional teams to implement new features, and troubleshoot technical issues across the stack. Other responsibilities may include writing and managing documentation, optimizing web performance, and supporting live application deployment processes.

Qualifications

  • Proficiency in front-end technologies such as HTML, CSS, JavaScript, and a front-end framework (e.g., React, Angular, or Vue.js).
  • Experience with back-end programming using frameworks like Node.js, Python, Ruby, or similar, as well as comfortable writing APIs.
  • Understanding of database systems, including SQL and NoSQL, and ability to design and query databases.
  • Familiarity with version control systems such as Git and basic knowledge of DevOps and deployment practices.
  • Excellent problem-solving, communication, and collaboration skills.
  • Working knowledge of cloud platforms like AWS, Azure, or Google Cloud is a plus.
  • Pursuing or recently completed a degree in Computer Science, Software Engineering, or a related field is preferred.
