About the Role

Frontend developer building modular components and micro-frontends.

About 360.agency

Human-sized, innovative agency delivering web solutions for automotive dealers.

Full Description

Vous rêvez d’une carrière ou votre expertise aura un impact dans une entreprise à taille humaine et innovante? Rejoignez l’équipe de 360.Agency! Nos 220 collaborateurs sont experts dans la création de solutions web et marketing conçues pour les concessionnaires automobiles canadiens.

Notre package comprend du fun, des horaires flexibles, une assurance collective active dès le premier jour, un service de télémédecine illimité pour vous et votre famille et plus encore!

Poste en mode hybride!

Nous cherchons un développeur Front-end pour contribuer au développement de notre plateforme de vente en ligne dédiée au secteur automobile.

Vous aimez relever des défis, explorer les meilleurs frameworks et technologies web, ce poste est fait pour vous!

……….
RESPONSABILITÉS

  • Développer et maintenir des composants modulaires front-end au sein d’une librairie utilisée par l’ensemble de nos produits;
  • Développer de nouveau modules applicatifs (micro-frontends) pour nos plateformes web;
  • Participer à l’évolution continue de nos plateformes applicatives;
  • Supporter et maintenir nos solutions actuelles.
  • Participation au processus de développement agile;
  • Programmation en pair, revue de code;
  • Collaboration avec les autres équipes (développeurs Back-end, intégrateurs, designers, gestionnaires de contenu, développeurs mobile)

……….
QUALIFICATIONS REQUISES

  • Fondamentaux Javascript + ES6
  • React (niveau intermédiaire)
  • Expérience de développement d’applications ou de modules applicatifs (SPA avec SSR)
  • Expérience avec les “States management” (Context API, Redux, etc.)
  • Expérience avec les récentes évolution de React (hooks, Suspense, etc.)
  • Node + npm (yarn)
  • Git ou système de versioning équivalent
  • HTML;
  • ​CSS/SCSS.
  • 3 ans d’expérience pertinente.

……….
ATOUTS

  • SASS / SCSS
  • Java / JSP
  • Spring & Spring Boot
  • Hibernate MySql
  • Elasticsearch, MongoDB, Redis
  • Serveurs applicatifs (Tomcat, Apache, Nginx)
  • Architecture REST et/ou orientée micro-service
  • Travail en contexte Cloud (Amazon Web Services ou équivalent)
