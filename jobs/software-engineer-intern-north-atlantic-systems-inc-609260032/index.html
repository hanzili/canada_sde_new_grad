Full Description

Overview
We are seeking a motivated Software Engineer Intern to join our dynamic development team. This internship offers an excellent opportunity to gain hands-on experience in software and application development across various technologies and platforms. The ideal candidate will have a passion for coding, problem-solving skills, and an eagerness to learn in a fast-paced environment. This paid internship provides exposure to real-world projects involving web development, cloud services, and modern development tools.

Responsibilities

  • Assist in designing, developing, and testing software applications using languages such as Java, C#, Python, C++, Ruby on Rails, and VBA.
  • Contribute to front-end development tasks utilizing JavaScript, HTML, CSS, and related frameworks.
  • Support backend development involving SQL databases like MySQL and SQL Server, including writing queries and optimizing database performance.
  • Collaborate on application development projects leveraging cloud platforms such as AWS and Azure.
  • Work with RESTful APIs and other integrations to enhance application functionality.
  • Participate in Agile development cycles, including sprint planning, stand-ups, and code reviews.
  • Use version control tools like GitHub, SVN, and Git for source code management.
  • Assist in troubleshooting issues within Linux or Windows environments and contribute to improving system stability.
  • Document technical specifications and contribute to team knowledge sharing.

Requirements

  • Currently enrolled in a Computer Science, Software Engineering, or related degree program or recent graduate.
  • Familiarity with web development frameworks and front-end technologies such as JavaScript, HTML/CSS.
  • Basic understanding of SQL databases (MySQL or SQL Server) and application development principles.
  • Exposure to cloud services like AWS or Azure is a plus.
  • Knowledge of REST APIs, GitHub/Git version control systems, and Agile methodologies preferred.
  • Programming experience in Java, C#, Python, C++, Ruby on Rails, or VBA is highly desirable.
  • Strong problem-solving skills with the ability to work collaboratively within a team environment.
  • Excellent communication skills and eagerness to learn new technologies. This internship is designed for individuals passionate about software development who are eager to build their skills while contributing to impactful projects in a collaborative setting.
