Software Engineering Internship

Full Stack Developer — Co-op/Intern

at Comercii Inc

📍 Location Kitchener, Ontario
📋 Details Co-op
📅 Posted Mar 13, 2026

About the Role

Build features, APIs, and data pipelines.

About Comercii Inc

Canadian AI-powered SaaS startup.

Full Description

Comercii is a Canadian technology company building an AI-powered SaaS platform that combines business directory discovery with managed website generation. Our platform serves 62+ industries across North America, with a data pipeline processing millions of business records. We're looking for a motivated co-op student to join our engineering team and contribute to real product development from day one.

The Role:

As a Full Stack Developer Intern, you'll work directly with the founder on core platform features — not side projects. You'll gain hands-on experience with modern web technologies, AI/ML pipelines, and production-scale data systems.

What You'll Work On

  • Build and maintain features across our Next.js/React/TypeScript web application

  • Develop and optimize data enrichment pipelines processing millions of business records

  • Integrate AI/ML tools for automated content generation and natural language search

  • Write clean, tested, production-ready code deployed to real users

  • Contribute to API development, database queries, and front-end UI components

What We're Looking For

  • Currently enrolled in a co-op program at a Canadian college or university

  • Familiarity with JavaScript/TypeScript and web development fundamentals

  • Interest in any of: React, Node.js, PostgreSQL, REST APIs, or data pipelines

  • Self-starter who can work independently in a remote environment

  • Strong communication skills and willingness to learn

Nice to Have (Not Required)

  • Experience with Next.js, Tailwind CSS, or React frameworks

  • Exposure to databases (SQL, PostgreSQL, or similar)

  • Interest in AI/ML, data processing, or automation

  • Personal projects or GitHub portfolio

What You'll Gain

  • Direct mentorship from the founder — no layers of management

  • Experience shipping real features to production, not mock exercises

  • Exposure to full-stack development, DevOps, AI integration, and SaaS architecture

  • A strong reference and portfolio piece for future employment

Tech Stack

  • Frontend: Next.js 16, React 19, TypeScript 5, Tailwind CSS 4

  • Backend: Node.js, PostgreSQL, Redis, Drizzle ORM

  • AI/ML: Ollama, Llama 3.1, NLP pipelines

  • Infrastructure: Railway, Vercel, Cloudflare, Docker

  • Tools: Git, GitHub, Turborepo, Vitest

How to Apply

Submit your resume and a brief note about what interests you about this role. Links to GitHub, personal projects, or a portfolio are a plus.

Job Type: Full-time

  • Work Location: In person
