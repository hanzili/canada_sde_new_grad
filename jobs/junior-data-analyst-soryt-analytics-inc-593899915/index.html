Full Description

**Job Summary** We are seeking a motivated Junior Data Analyst to join our dynamic team. This entry-level position offers an excellent opportunity to develop your data analysis skills and contribute to impactful projects. The ideal candidate will support data-driven decision-making processes by analyzing complex datasets, creating visualizations, and assisting with project management tasks. A strong foundation in analytics tools and programming languages, along with a keen eye for detail, is essential for success in this role.

**Duties**

- Collect, clean, and analyze large datasets using SQL, R, Python, and other data analysis tools - Develop interactive dashboards and visualizations utilizing Tableau and Power BI to communicate insights effectively - Support business analysis by identifying trends, patterns, and opportunities within data sets - Assist in database design and management using Microsoft SQL Server and Oracle databases - Collaborate with cross-functional teams following Agile methodologies to ensure timely project delivery - Contribute to the Software Development Life Cycle (SDLC) by documenting requirements and testing solutions - Create detailed process diagrams and documentation using Visio for project workflows - Automate repetitive tasks through VBA scripting and Bash (Unix shell) scripting where applicable - Participate in project management activities to help coordinate tasks and meet deadlines

**Skills**

- Proficiency in data visualization tools such as Tableau and Power BI - Strong knowledge of SQL, including experience with Microsoft SQL Server and Oracle databases - Programming skills in R and Python for data analysis and modeling - Familiarity with SDLC processes, Agile project management, and business analysis principles - Experience with database design and data warehousing concepts - Ability to analyze complex datasets using analytics techniques and analysis skills - Knowledge of VBA scripting for automation tasks - Basic understanding of Bash (Unix shell) scripting for data processing automation - Excellent problem-solving skills with attention to detail - Effective communication skills for presenting insights clearly to stakeholders

This position provides an exciting opportunity for aspiring data analysts to grow their expertise within a collaborative environment. Candidates should demonstrate a passion for data analytics, continuous learning, and a proactive approach to problem-solving.
