As an AI Engineering Intern on the Industrial AI team, you will help build and productionize LLM-powered capabilities for DI product technical support. You’ll work hands-on across the GenAI stack - RAG, prompt and context design, automated evaluation, LLMOps monitoring, and data refinement - to improve groundedness, reduce hallucinations, and ensure reliable performance in an enterprise Azure environment. You’ll collaborate with engineers and domain experts to ship measurable quality improvements for English and French user queries. You’ll also contribute to thought leadership by creating content and supporting the company’s overall GenAI strategy and adoption initiatives.

What You’ll Do

  • Research and learn about various GenAI tools, platforms, and use cases relevant to Siemens' business
  • Build and iterate on RAG pipelines (chunking, retrieval strategies, citation/grounding behavior) to improve answer accuracy for technical support content
  • Develop and maintain prompt/context templates and lightweight guardrails to increase consistency and reduce hallucinations
  • Design and implement evaluation harnesses (golden test sets, regression tests, automated metrics) to measure quality across English and French queries; coordinate spot checks with French-speaking SMEs
  • Run structured experiments (e.g., retrieval/prompt variants) and analyze results to recommend changes based on quality, latency, and cost
  • Implement LLMOps/observability: logging, dashboards, and monitoring for answer quality signals, failure modes, latency, and token/cost usage
  • Integrate user feedback mechanisms (thumbs up/down, failure tagging) and close the loop with systematic improvements
  • Support deployment and operations in Microsoft Azure (CI/CD, environments, , secure access patterns) in collaboration with the engineering team
  • Document best practices and contribute to internal enablement material (usage guidelines, prompting tips, quality playbooks)

What You’ll Bring

  • Currently enrolled in a Master’s program in computer science, engineering, AI, or a related technical field.
  • Solid software engineering skills with experience building production-grade code (Python preferred, plus REST APIs, Git, Docker)
  • Hands-on experience with LLMs in applied/production settings, such as building GenAI apps using RAG, tool/function calling, and prompt/context engineering
  • Familiarity with LLM evaluation concepts (test sets, regression testing, groundedness/faithfulness, qualitative error analysis); experience with evaluation frameworks is a plus
  • Experience with cloud development - Microsoft Azure is a strong plus (Azure AI, monitoring/logging, CI/CD)
  • Strong analytical skills: able to translate ambiguous quality issues into measurable metrics and experiments
  • Good communication and collaboration skills across engineering and domain stakeholders
  • French is an asset but not required (French-speaking SMEs will support validation)

Hourly rate is commensurate with experience, and ranges between $ 22.70 CAD - $ 28.00 CAD.
