RQ10660 - Software Developer - Junior

at S M Software Solutions Inc

📍 Location Toronto, Ontario
📋 Details Contract · 10mo · May 2026–Mar 2027
📅 Posted Apr 25, 2026

About the Role

Develop/test web apps with C#/Angular

About S M Software Solutions Inc

Ontario government ministry; public service delivery

Full Description

We are seeking an immediate Software Developer - Junior for the Ministry of Public and Business Service Delivery and Procurement in an onsite role (5 days/week) at 777 Bay St, 3rd Fl, Toronto, Ontario. This junior developer builds/tests/deploys web applications using C#/.NET 4.8+, Angular/AngularJS, Oracle 19c PL/SQL, Azure DevOps/Git/YAML pipelines, and ensures AODA/WCAG AA accessibility compliance while supporting SDLC activities including troubleshooting, unit testing, and Oracle E-Business Suite customizations.

Requisition Class: Not specified

Job Title: RQ10660 - Software Developer - Junior

Client: Ministry of Public and Business Service Delivery and Procurement

Work Location: 777 Bay St, 3rd Fl, Toronto, Ontario (Onsite)

Estimated Start Date: May 4, 2026

Estimated End Date: March 31, 2027

Business Days: 220 (Extension: Probable)

Hours per Day/Week: 7.25 hours per day

Security Level: No Clearance Required

Engagement Type: Contract / Consulting

Key Responsibilities Software Development (45%)

  • Develop/test/debug web applications across browsers using Visual Studio 2017/2022.
  • Build reusable components; analyze requirements for reports/system enhancements.
  • Troubleshoot performance issues in web apps/Windows services without server access.
  • AODA/WCAG AA compliance for accessible web applications.

Technical Skills (40%)

  • Backend: C#/.NET Framework 4.8+, Oracle 19c PL/SQL, XML.
  • Frontend: Angular v17 (TypeScript/RxJS/NGRX), AngularJS, Bootstrap, JS/jQuery/HTML5/CSS3.
  • DevOps: Azure DevOps (Git/YAML pipelines), Windows/Linux admin.
  • Relational database integration; automated build/deployment pipelines.

Soft Skills (10%)

  • Collaborate with QA/product owners/technical stakeholders via clear communication.
  • Agile/Scrum team environment; learn new technologies quickly.
  • Problem-solving, attention to detail, meet deadlines.

Technical Areas in Scope Stack: C#/.NET 4.8, Angular 17/AngularJS, Oracle 19c PL/SQL, Azure DevOps/Git.

Domains: Web apps, Oracle EBS customizations (GL/AR/AP/INV), AODA/WCAG AA.

Tools: Visual Studio 2017/2022, YAML pipelines, Bootstrap/jQuery.

Must-Haves

  • C#/.NET 4.8+ with Visual Studio 2017/2022 experience.
  • Oracle 19c PL/SQL; XML processing.
  • Angular (v17 TypeScript/RxJS/NGRX), AngularJS, modern web tech.
  • Azure DevOps Git/YAML pipelines; AODA/WCAG AA knowledge.

Desired Skillset

  • Oracle EBS Financial modules (GL/AR/AP/INV).
  • SQL performance tuning; data warehousing/ETL/OBIEE.
  • Java/Oracle ADF; Unix shell scripting.

Engagement Requirements

  • Location: Onsite 5 days/week, Toronto (777 Bay St).

Deliverables

  • Working application components, unit tests, documentation.
  • System enhancements, bug fixes, release support.
  • AODA-compliant web applications, Oracle integrations.

How To Apply Submit by Monday, April 27, 2026, 11:30 AM EST to kenny.offor@smsoftconsulting.com OR hrsmss@smsoftconsulting.com.

  • References
  • Expected hourly rate

  • LinkedIn ID
