Full Description

Software Developer Co-op Overview 

Are you interested in designing and implementing a scalable, flexible architecture on PlayStation®? Our Software Engineering Co-ops work alongside our full time software engineers to produce real products and services that matter to our customers. Focusing on PS5 console software, you’ll contribute to developing features that enhance the PlayStation® experience for millions of users. Collaborating with a team of skilled engineers, you’ll help build modern API-driven applications for the PlayStation®5 platform. As part of a globally distributed team, you’ll gain valuable experience in a dynamic, multicultural environment while implementing innovative console software solutions.  

We’re part of PlayStation’s CAPE organization, building the systems and experiences that help players discover, purchase, and engage with content across the PlayStation ecosystem. Our work supports millions of players worldwide, focusing on making every interaction simple, seamless, and enjoyable. We’re hiring across two areas: Transact, which powers purchase and payments, and Nurture (Merchandising & Experiences), which focuses on personalized content discovery and dynamic user experiences. You’ll work on high-scale systems, collaborate across Sony, and help shape how players connect with games every day.

Responsibilities: 

  • Produce well written features that will be used by millions of passionate customers

  • Help maintain existing applications

  • Write clear, maintainable, portable, and highly functional code

  • Test and document code produced

Minimum Qualifications:

  • A good understanding of object-oriented principles.

  • Familiarity with common data structures and algorithms

  • Fluent in Python

  • Experience with some web technologies and frameworks

  • Experience developing single-page web applications using frameworks like Ember.js, React Native, AngularJS, Backbone.js

  • Experience with Customer-facing UI Development

Preferred Requirements:

  • Proficient with JavaScript

  • Knowledge of HTML5 and CSS3 features and modern browser architecture

  • Experience with agile development practices

  • Experience with Node.js and Redis, or similar technologies

  • Experience with C#, Java, or OO C++ code is recommended

  • Proven track record of shipping applications (not just static websites) built on modern web technologies

  • Excellent communication skills, and hopefully a good sense of humor!
