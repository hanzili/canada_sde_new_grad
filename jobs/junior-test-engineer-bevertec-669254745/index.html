Full Description

Title: QA Specialist

Duration: Till March 2027 (possible extension)

Location: Toronto

Domain: Government Services

Salary: $40-$50 per hour (based on experience)

Assignment Type - Fully Onsite

Must have:

  • Strong manual testing skills to complement automated testing and ensure comprehensive test coverage.
  • Experience creating, executing, and maintaining automated test scripts using tools such as Selenium or Playwright .
  • Basic knowledge of writing and troubleshooting test scripts, with familiarity in JavaScript or TypeScript for testing web applications (e.g., Angular-based applications).
  • Experience querying relational databases (e.g., Oracle) using SQL to validate test results; familiarity with PL/SQL is an asset.
  • Experience using Azure DevOps to manage test cases, log and track defects, and support test execution.
  • Experience testing RESTful APIs using tools such as Postman or SoapUI .
  • Familiarity with CI/CD pipelines, including integrating or executing automated tests within Azure DevOps.
  • Experience or exposure to accessibility testing for web applications, including knowledge of WCAG guidelines and use of automated tools (e.g., AXE) and screen readers (e.g., NVDA, Narrator).
  • Experience testing reports and enhancements to existing systems.
