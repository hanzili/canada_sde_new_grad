About Tartigrade

Full Description

Job Description

We are looking for an intern to develop, deploy, and maintain embedded environmental monitoring systems for a rural IoT site. You will work on ESP32-based sensor nodes, field hardware installation, Home Assistant integration, Cloudflare networking, and system reliability improvements.

Responsibilities

Sensor & Hardware Systems

  • Install, calibrate, and maintain environmental sensors (temperature, flow, ultrasonic, septic level, TDS, turbidity).
  • Build and integrate hardware with ESP32 microcontrollers using ESPHome.
  • Assemble robust field enclosures (IP-rated boxes, cable glands, waterproof routing).
  • Diagnose wiring issues, sensor noise, power instability, and field connectivity problems.

Embedded Firmware

  • Develop and update ESPHome YAML configurations for ESP32 devices.
  • Design basic circuits and interface sensors to microcontrollers.
  • Ensure reliable measurement accuracy and long-term field operation.

Home Assistant Integration

  • Configure sensors, dashboards, entities, automations, and Elasticsearch charts.
  • Manage the Home Assistant server (Ubuntu + VirtualBox).
  • Maintain long-range remote access via Cloudflare Tunnel.

Networking & Remote Access

  • Configure Cloudflare Zero Trust tunnels for Home Assistant, Linux hosts, and Shinobi CCTV.
  • Maintain DNS routing, config.yml files, tunnel credentials, and systemd services.
  • Set up static IPs with Netplan YAML and ensure stable Ethernet-based networking.

Camera System Engineering

  • Deploy LAN-only surveillance systems using Shinobi.
  • Configure ONVIF/RTSP cameras and low-power Pi Zero 2W tunnel endpoints.
  • Maintain a secure, cloud-independent camera network.

Documentation & Team Support

  • Maintain sensor indexes, installation guides, and network documentation.
  • Support capstone project teams with wiring, integration, and embedded system guidance.

Requirements

Must-Have Skills:

  • Experience with ESP32 microcontrollers and embedded sensor integration.
  • Familiarity with ESPHome, YAML, and Home Assistant.
  • Basic circuit design, wiring, soldering, and hardware assembly.
  • Networking fundamentals (DNS, static IPs, tunneling, systemd services).
  • Linux/Ubuntu proficiency for IoT service deployment.
  • Troubleshooting embedded hardware, sensors, and Wi-Fi issues.

Nice-to-Have:

  • ONVIF/RTSP camera experience.
  • PCB prototyping or electronics fabrication.
  • Elasticsearch or dashboard database configuration.
  • Raspberry Pi setup and optimization.

Eligibility:

  • Enrolled or graduated from electrical engineering, software engineering, computer science, or a closely related program.
  • Eligible to work in Canada.

Job Types: Full-time, Permanent

Pay: From $75,000.00 per year
