About the Role

About Nokia

Full Description

Job Description Duration: 2 Months

Date: July-August 2026

Location: Onsite (Ottawa campus)

How You Will Contribute And What You Will Learn

  • Develop software solutions encompassing specific functions, automation, or custom tools
  • Responsible for own code quality, review, testing, troubleshooting, and debugging activities
  • Troubleshoot and problem-solve software bugs, working closely with the engineering teams to resolve issues
  • You will be exposed to one or more of the following technologies: UI/UX development in React, Scripting and app development in Python, Java, JavaScript, GoLang, Deployment and automation tools like Terraform and Ansible, Test automation development frameworks such as Radish, and Cloud services and deployments (GCP, Azure).

Key Skills And Experience

  • Currently a high school student with a passiona for tech.
  • Programming experience in one or more of Python, Java, JavaScript, BASH, GoLang.
  • Familiarity with common development tools such as Git, Jira, Confluence, and Jenkins.
  • Exposure to Virtualization and Cloud technologies, along with knowledge of Containerization tools like Docker, Kubernetes, and Helm

It would be nice if you also had:

  • Strong knowledge of basic IP networking concepts
  • Familiarity with Linux distros.
  • Familiarity with Continuous Integration and Agile development processes.
  • Understanding of OS and system software concepts, including processes, threads, memory
  • management, and synchronization, and exhibits excellent debugging skills.
