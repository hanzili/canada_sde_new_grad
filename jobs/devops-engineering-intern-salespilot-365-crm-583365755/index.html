Full Description

About SalesPilot 365

SalesPilot 365 is a modern staffing and talent acquisition platform dedicated to helping leading organisations connect with exceptional talent. As we expand our presence in the AI/ML and Cloud Engineering space, we are seeking motivated, analytical, and curious individuals to join our DevOps Engineering Internship Cohort .

Role Overview

As a DevOps Engineering Intern at SalesPilot 365 , you will work closely with our engineering and infrastructure teams to learn and apply industry-standard practices related to automation, cloud infrastructure provisioning, continuous integration and deployment (CI/CD), and system monitoring. This hands-on internship is designed for enthusiastic learners who are eager to build core DevOps skills from the ground up. No prior experience is required.

Key Responsibilities

  • Learn to set up and manage CI/CD pipelines using tools such as Jenkins, GitHub Actions, or GitLab CI.
  • Assist in containerising applications using Docker and deploying them within Kubernetes environments.
  • Support infrastructure automation using Infrastructure-as-Code (IaC) tools such as Terraform or AWS CloudFormation.
  • Monitor system health, performance, and availability using tools like Prometheus, Grafana, and the ELK Stack.
  • Develop automation scripts using Python or Bash to streamline operational tasks.
  • Collaborate with development teams to troubleshoot deployment and build issues.
  • Assist in configuring and managing cloud services across AWS, Azure, or GCP for scalable and efficient deployments.
  • Gain experience with configuration management tools such as Ansible or Chef.

Requirements

  • Currently pursuing or recently completed a B.Tech/BE/M.Tech/MSc in Computer Science, IT, Information Systems, or a related field.
  • Basic understanding of operating systems (Linux preferred).
  • Familiarity with version control systems, especially Git.
  • A strong desire to learn DevOps methodologies, cloud technologies, and automation tools from scratch.
  • Problem-solving mindset, curiosity, and keen attention to detail.
  • Good communication skills, both written and verbal.

Nice to Have (Bonus Skills)

  • Basic experience with Python or Bash scripting.
  • Familiarity with cloud platforms such as AWS, Azure, or GCP.
  • Introductory understanding of containers (Docker).
  • Awareness of CI/CD pipelines or Infrastructure-as-Code principles.

What You‚Äôll Gain

  • Practical exposure to modern DevOps workflows and cloud infrastructure used by leading technology organisations.
  • Hands-on experience managing CI/CD pipelines, automating deployments, and monitoring system performance.
  • 1:1 mentorship from experienced DevOps engineers.
  • Internship Certificate and Letter of Recommendation upon successful completion.
  • Potential opportunity for a Pre-Placement Offer (PPO) with SalesPilot 365 or partner client companies.

  • Starting stipend of C$20 per hour , with the possibility of a higher rate based on performance in assessments and interviews.
