About BorlaSoft Solutions Inc.

Edmonton-based internship with cloud tech exposure.

Full Description

Job Summary
We are seeking a motivated Software Engineer Intern to join our dynamic development team. This internship offers an excellent opportunity to gain hands-on experience in software development, web applications, and cloud technologies. The ideal candidate will be passionate about coding, eager to learn, and interested in contributing to innovative projects across various platforms and languages.

Duties

  • Assist in designing, developing, and testing web applications using technologies such as JavaScript, Ruby on Rails, and C#
  • Support application development efforts utilizing SQL databases like MySQL and SQL Server, ensuring efficient data management
  • Contribute to software development projects involving languages such as Java, Python, C++, and C#
  • Collaborate with team members using version control systems like GitHub, SVN, Git, and Azure DevOps
  • Participate in Agile development processes including sprint planning and daily stand-ups
  • Help integrate RESTful APIs and third-party services into existing applications
  • Support deployment and maintenance of applications on cloud platforms such as AWS and Azure
  • Assist in troubleshooting issues on Linux-based systems and contribute to front-end development tasks
  • Document code changes and technical specifications for ongoing project continuity

Requirements

  • Currently pursuing a degree in Computer Science, Software Engineering, or a related field
  • Strong understanding of web development frameworks and front-end technologies including JavaScript and front-end libraries
  • Familiarity with application development using languages like Java, Python, C++, or C#
  • Experience working with SQL databases such as MySQL or SQL Server; knowledge of database design is a plus
  • Basic understanding of cloud services including AWS or Azure is preferred
  • Knowledge of REST APIs, GitHub, SVN, Git version control systems, and Agile methodologies is desirable
  • Familiarity with Linux operating systems and scripting is advantageous
  • Ability to work collaboratively in a team environment with excellent communication skills
  • Eagerness to learn new technologies and adapt quickly to changing project requirements

This internship provides valuable industry experience while working on impactful projects within a supportive team environment.
