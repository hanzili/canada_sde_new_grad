Junior Web Developer

at mynexttrip.ai

📍 Location Vancouver, British Columbia Remote

About the Role

About mynexttrip.ai

AI-powered travel platform startup

Full Description

**Salary: 80k

Company Description**

At My Next Trip, we are developing mynexttrip.ai, an AI-powered travel platform designed to redefine how people plan and enjoy their journeys. Our technology seamlessly integrates intelligent flight and hotel bookings, personalized city exploration, and tailored full itineraries. With features that assist users in discovering attractions, booking tickets, and finding popular local restaurants, we deliver a smart, all-in-one travel experience.

Role Description

This is a full-time, remote role for a Junior Web Developer. The role includes supporting both front-end and back-end web development, contributing to the design and functionality of our platform, and writing efficient, clean, and scalable code. The Junior Web Developer will collaborate with a multidisciplinary team, actively participate in troubleshooting, and contribute to feature enhancements for an engaging user experience.

Qualifications

  • Proficiency in Front-End Development, including using HTML, CSS, and JavaScript to create engaging user interfaces
  • Experience with Back-End Web Development, such as server-side programming and database management
  • Familiarity with general Web Development and Web Design principles
  • Strong programming skills and an ability to write clean, maintainable code
  • Good problem-solving abilities and a desire to learn and grow in a collaborative team environment
  • Knowledge of responsive design and cross-platform compatibility
  • Bachelor’s degree in Computer Science, Information Technology, or a related field, or equivalent practical experience
  • Understanding of or interest in travel and technology is a plus
