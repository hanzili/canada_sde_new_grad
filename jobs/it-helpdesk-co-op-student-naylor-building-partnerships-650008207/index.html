IT Helpdesk Co-op Student

at Naylor Building Partnerships

📍 Location Oakville, ON L6H 1A5
📋 Details Co-op · 4mo · Summer 2026
📅 Posted Mar 24, 2026

About the Role

Provide desktop and application support to users

About Naylor Building Partnerships

Ontario-based building partnerships firm

Full Description

Naylor Building Partnerships Inc. is currently recruiting for an IT Helpdesk - Co-op Student to join our IT team for the May to August 2026 Summer work term. The successful candidate must be in an accredited co-operative education program, and be able to work full-time from our Oakville office location (not a remote role):

Responsibilities:

  • Provide desktop/application support to employees, both onsite and in remote offices/branches
  • Assist users by answering software and hardware related questions
  • Evaluate the criticality of problems by effectively troubleshooting and, when necessary, escalate issues
  • Assist with maintenance of hardware standards for laptops, desktops and printers
  • Maintain existing, and add new printers to the network
  • Configure desktops and laptops, as required
  • End user support of application software including MS Office, VPN Client, etc.

Requirements:

  • Relevant Computer Systems/ Engineering Technology diploma/degree in progress
  • Exposure to relevant courses such as computer programming languages, Windows OS, basic networking concepts, database and systems development considered an asset
  • Understanding of Linux Operating Systems considered an asset
  • Solid problem-solving skills
  • Detail oriented with strong communication skills
  • Strong knowledge of MS office specifically, Microsoft Excel, Word and Outlook.
  • Willingness to take on other duties as needed

Please note: this posting is for an existing vacancy.
