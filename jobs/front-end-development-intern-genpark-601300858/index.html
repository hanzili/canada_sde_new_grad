Full Description

**Company Description**

GenPark leverages intelligent Brand Agents and User Agents powered by AI to help global brands effectively engage and convert Gen Z consumers. Through personalized AI-generated content, interactive Agents, and immersive video experiences, GenPark empowers brands to connect with their target audience in meaningful ways. **Role Description**

This internship opportunity is for a Front End Development Intern at GenPark. The intern will assist in designing, developing, and debugging front-end components and interfaces to enhance user experiences. Key responsibilities include collaborating with the development team, implementing responsive web designs, testing and troubleshooting applications, and writing clean and maintainable code. This is a hybrid role based in Vancouver, BC, allowing flexibility with some remote work.
