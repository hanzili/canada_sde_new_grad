Full Description

Embedded System Intern or coop student. Able to further develop and enhance our “Spectrum” Mass Spec application to support on-going requirements, including the capability to control NexION ICP-MS mass spectrometer and to be able to run customizable test scripts. When used as an instrument or PCB test application, it must run customizable test scripts and generate test reports. Requires C/C++, Rust and JavaScript coding skills. The role will involve collaboration with research scientists and the electrical and manufacturing teams within PerkinElmer
