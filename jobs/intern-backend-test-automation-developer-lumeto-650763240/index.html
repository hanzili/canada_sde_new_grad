About Lumeto

Lumeto’s immersive simulation lab, InvolveXR, aims to meaningfully reduce the social and financial cost of medical error. Through flexible, virtual simulated environments, we help improve clinician preparedness, confidence, and decision-making across the healthcare ecosystem.

The Role

Reporting to the engineering team, the Backend / Test Automation Intern will support the development and maintenance of automated testing systems and backend validation processes. This role is ideal for someone who enjoys scripting, debugging, and working across APIs and backend systems. You’ll contribute to automation frameworks and help improve the reliability and scalability of our platform.

This is a full-time, paid internship (12-16 weeks minimum), operating on an EST schedule.

Responsibilities

  • Develop and maintain backend QA automation scripts (e.g., Postman or similar tools)
  • Perform integration and API testing across services (REST, GraphQL, gRPC)
  • Debug, troubleshoot, and resolve issues across backend systems
  • Contribute to test automation frameworks and improve test coverage
  • Collaborate with engineering teams to validate system performance and reliability
  • Support CI/CD and testing workflows within cloud environments

Qualifications

Must-Haves

  • 1–2 years of experience or academic exposure in backend development or QA automation
  • Experience with scripting or programming (Python, C#, or similar)
  • Familiarity with API testing (Postman or equivalent)
  • Understanding of REST, GraphQL, or gRPC APIs
  • Strong problem-solving and debugging skills
  • Ability to manage multiple tasks and work within deadlines
  • Currently pursuing or recently completed post-secondary education (3rd year+ / CEGEP or equivalent)

Nice-to-Haves

  • Experience with Azure (preferred), AWS, or GCP
  • Exposure to Infrastructure as Code (IaC)
  • Experience with Azure DevOps or CI/CD pipelines
  • Familiarity with UI or acceptance test automation
  • Experience using AI tools to support development workflows

What You’ll Gain

  • Real-world experience building and scaling QA automation systems
  • Exposure to cloud infrastructure and modern backend architectures
  • Opportunity to work with a cutting-edge XR platform in healthcare
  • Mentorship from experienced engineers in a fast-paced environment
