Eligibility (MANDATORY):

  • Graduated within the last 2 years

About the Role
We are developing a vision-based prototype that requires controlled, real-world testing. This hybrid role includes regular on-site sessions to configure lighting, camera setups, and validate system behavior.

Responsibilities

  • Set up hardware testing environments (lighting, camera placement, calibration)
  • Run computer vision experiments and analyze results
  • Clean and organize datasets
  • Implement or adjust CV inference pipelines
  • Conduct live trials on site and document findings

Required Skills

  • Experience with computer vision (OpenCV, PyTorch, or similar)
  • Ability to set up and optimize physical testing environments
  • Familiarity with cameras, sensors, or robotics (assets)
  • Git/GitHub proficiency
  • Ability to work independently and troubleshoot on site

Assets

  • Experience with embedded cameras or robotics
  • Experience optimizing lighting for machine vision
  • Familiarity with GPU workflows (CUDA/AWS)
