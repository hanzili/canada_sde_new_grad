Full Description

Junior Full Stack Engineer (Python/Azure/Mobile)

Location: Montreal, Quebec (Hybrid)

Job Type: Full-Time Contract

Company: Plūme Inc.

About the Role

We are a forward-thinking law firm in Montreal seeking a Junior Full Stack Engineer to take ownership of our proprietary AI-driven HR Management platform. The application is currently live (iOS, Android, and Web), and we are transitioning development in-house to scale, optimize, and evolve the product.

This is a unique opportunity for a skilled developer to act as the technical lead for an existing product. You will work directly with the Partner to debug issues, refactor legacy code, and build new AI-driven features.

Key Responsibilities

Codebase Ownership: Audit, maintain, and improve the existing source code inherited from external vendors.

Web Development (TypeScript): Manage and expand the web application using TypeScript to ensure type safety and a robust user experience.

Mobile Maintenance: Debug and deploy updates for our iOS (Swift) and Android applications.

Back-End Engineering: Manage and scale the existing back-end services on Python and Microsoft Azure.

Feature Development: Design and code new modules, specifically focusing on AI/Machine Learning integration.

Problem Solving: Identify bugs, troubleshoot performance bottlenecks, and implement long-term fixes independently.

Qualifications

Technical Proficiency: Strong experience with Python (Back-End) and TypeScript (Front-End) is required.

Cloud Experience: Familiarity with Microsoft Azure services.

Mobile Development: Exposure to Swift (iOS) and Android development environments.

Forensic Coding: Ability to read, understand, and fix code written by other developers.

Location: Must be based in the Greater Montreal area (Hybrid).

Preferred Skills (Nice to Have)

Experience with AI/Machine Learning libraries.

Experience transitioning projects from external contractors to internal management.

Why Apply?

Immediate Impact: You aren't just writing code in a void; you are improving a live product used by real people.

Autonomy: Enjoy a high degree of independence in how you solve technical challenges.

Growth: A rare chance for a junior developer to act as a technical lead on a project with startup agility and law firm stability.

Responsabilités principales

  • Prise en charge du code source : Auditer, maintenir et améliorer la base de code existante héritée de fournisseurs externes.
  • Développement Web (TypeScript) : Gérer et faire évoluer l’application Web en TypeScript afin d’assurer la sécurité des types et une expérience utilisateur robuste.
  • Maintenance mobile : Déboguer et déployer des mises à jour pour les applications iOS (Swift) et Android.
  • Développement back-end : Gérer et faire évoluer les services back-end existants en Python et sur Microsoft Azure.
  • Développement de fonctionnalités : Concevoir et coder de nouveaux modules, avec un accent particulier sur l’intégration de l’IA et de l’apprentissage automatique.
