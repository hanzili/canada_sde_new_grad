About the Role

Build Kotlin apps, REST APIs, deploy to cloud

About TekFormix

Mississauga-based tech firm delivering software solutions

Full Description

Duties

  • Develop and maintain robust applications using Kotlin, ensuring optimal performance and scalability.
  • Collaborate with cross-functional teams to design and implement RESTful APIs and integrate third-party services.
  • Contribute to the full software development lifecycle, including planning, coding, testing, deployment, and maintenance.
  • Work with IT infrastructure components such as AWS, Azure, Linux, and SQL databases like MySQL to support application deployment and management.
  • Participate in Agile development processes, including sprint planning and daily stand-ups.
  • Utilize version control systems such as GitHub, SVN, and Git for source code management.
  • Engage in front-end development tasks using JavaScript, React Express, and related technologies when necessary.
  • Ensure application security through RBAC (Role-Based Access Control) and other best practices.
  • Write clean, efficient code adhering to industry standards and best practices in software engineering.
  • Assist in troubleshooting issues related to application performance or infrastructure components like C#, C++, Python, VBA, and .NET frameworks.

Requirements

  • Proven experience in software development with a strong focus on Kotlin programming language.
  • Solid understanding of IT infrastructure including AWS, Azure, Linux environments, and cloud-based deployment strategies.
  • Familiarity with web development technologies such as JavaScript, React Express, REST APIs, and RESTful architecture principles.
  • Experience working with databases like MySQL or SQL Server; knowledge of application development involving C#, C++, Python is a plus.
  • Proficiency with version control systems including GitHub, SVN, and Git.
  • Knowledge of Agile methodologies for efficient project management and delivery.
  • Understanding of security protocols such as RBAC to ensure secure application access controls.
  • Exposure to other programming languages like Java or C# is advantageous.
  • Experience working on complex projects involving API integrations or cloud infrastructure is preferred.
  • Strong problem-solving skills with the ability to adapt quickly in a fast-paced environment. Join our team as a Kotlin Developer to contribute your expertise toward building innovative solutions that impact our clients worldwide while advancing your career in a supportive professional setting.
