Position Summary We are seeking a curious and motivated Automation QA Intern to join our QA Engineering team for a co-op term. This role offers hands-on experience in writing and testing automation scripts for real-world software projects. You’ll work closely with QA engineers and developers in a collaborative, DevOps-driven environment that values learning, mentorship, and quality.

This internship is designed to support students who are building foundational skills in automation and quality assurance, offering mentorship and exposure to modern QA practices.

Key Responsibilities

  • Develop and maintain basic automated test scripts for web and API testing using tools such as Selenium, Playwright, or Postman, with guidance from QA engineers.
  • Execute test scripts to validate new features and bug fixes across multiple environments.
  • Log and track defects found during testing and collaborate with QA engineers and developers to troubleshoot and resolve issues.
  • Review project requirements and test cases to understand testing scope and identify automation opportunities.
  • Assist in updating and organizing test documentation, including test cases, test data, and test results.
  • Learn and apply version control practices using Git to manage and update test scripts.
  • Participate in team meetings and various ceremonies to gain exposure to Project and DevOps workflows.
  • Support QA engineers in integrating automated tests into CI/CD pipelines (e.g., GitHub Actions, Jenkins).
  • Actively seek feedback, ask questions, and contribute to a culture of continuous learning and quality improvement.

Required Qualifications

  • Currently enrolled in a Computer Science, Software Engineering, or related undergraduate program.
  • Basic understanding of programming concepts (e.g., C#, JavaScript, Python, or Java), gained through coursework or personal projects.
  • Hands-on experience writing simple scripts or test cases—through labs, assignments, or personal exploration.
  • Ability to write, read, and debug simple code with support from the team.
  • Familiarity with object-oriented programming principles and basic understanding of test frameworks or unit testing.
  • Interest in software testing, automation, and quality assurance.
  • Strong attention to detail and willingness to learn.
  • Effective communication and collaboration skills.

Preferred Qualifications

  • Exposure to automation tools or frameworks (e.g., Selenium, Playwright).
  • Familiarity with Git or other version control systems.

Learning Outcomes

  • Gain practical experience in automation testing and QA processes.
  • Work with enterprise-level tools and CI/CD practices.
  • Receive mentorship from experienced QA professionals.
  • Contribute to meaningful projects that impact product quality.
