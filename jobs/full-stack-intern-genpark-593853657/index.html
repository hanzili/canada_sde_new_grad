**Company Description**

GenPark is revolutionizing the retail landscape by introducing emerging Oriental direct-to-consumer brands to a global Gen Z audience. **Role Description**

This is a hybrid internship role for a Full Stack Developer Intern, based in Vancouver, BC, with opportunities for remote work. The intern will assist in designing, developing, and maintaining web applications and features. Typical responsibilities include writing clean, scalable, and efficient code for both front-end and back-end systems, collaborating with the team to define project requirements, and troubleshooting bugs or issues. Additional tasks include implementing APIs, supporting database management, and improving the performance of existing applications.

**Qualifications**

- Proficiency in front-end technologies such as HTML, CSS, and JavaScript, with experience in modern frameworks like React or Angular. - Knowledge of back-end development, including languages like Node.js, Python, or Ruby, and experience working with server-side frameworks. - Familiarity with database management systems, such as MySQL, PostgreSQL, or MongoDB. - Understanding of RESTful APIs and experience integrating APIs into web applications. - Problem-solving and debugging skills to identify and resolve issues quickly. - A collaborative mindset and strong communication skills for teamwork in a hybrid environment. - Currently pursuing or recently completed a degree in Computer Science, Software Engineering, or a related field. - Knowledge of cloud platforms, such as AWS, Google Cloud, or Azure, and version control tools (e.g., Git) is a plus.
