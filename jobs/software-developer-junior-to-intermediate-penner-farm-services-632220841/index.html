About the Role

Develop, test, and deploy software for barn systems

About Penner Farm Services

Agritech firm building barn-control software

Full Description

Join Cortex Agritechnology and be part of a new era in farming — where innovation meets tradition to transform how we cultivate the land and feed the world. Using cutting-edge data analytics, sensors, and artificial intelligence, we’re driving a more sustainable, efficient, and productive agricultural future.

We are seeking a Junior/Intermediate Software Programmer with a passion for customizing and improving existing software solutions. The successful candidate will work closely with our engineering team to adapt and enhance our barn control systems, which manage livestock environments and equipment. This is a hands-on role where you will gain valuable experience in applying software development skills to real-world agricultural technology projects.

The position will be responsible for duties such as:

  • Customize and enhance existing software applications on a project-by-project basis.
  • Develop, test, and maintain code in .NET, Python, Structured Text, and Function Block programming.
  • Collaborate with engineers and project managers to implement customer-specific requirements.
  • Assist with integrating new features into existing barn control systems.
  • Participate in debugging, troubleshooting, and optimizing system performance.
  • Utilize version control (GitHub) and containerization tools (Docker) to support development workflows.
  • Document code changes, project configurations, and technical processes.
  • Provide support during system testing and deployment in agricultural environments.
  • Provide basic I.T. support for connecting systems together

Skills Required:

  • Diploma or degree in Computer Science, Software Engineering, or a related field (or equivalent practical experience).
  • Some experience with: NET programming, Python, Structured Text and Function Block programming (IEC 61131-3), Networking and basic communication (Ethernet IP, Modbus), Basic terminal commands on Linux and Windows, PLC programming (Beckhoff PLC experience an asset)
  • Familiarity with GitHub and Docker is considered an asset.
  • Strong problem-solving skills and ability to work independently and in a team environment.
  • Willingness to adapt existing codebases and tailor them to meet customer needs.
  • Enthusiasm for learning and applying software in real-world control system applications.
  • Knowledge of or experience in livestock agriculture is an asset.

Cortex Agritechnology offers a competitive wage along with a comprehensive health and dental benefits plan. We offer an excellent work environment with opportunities for development and advancement.
