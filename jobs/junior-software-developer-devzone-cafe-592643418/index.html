Full Description

Job Overview
We are seeking a motivated Junior Software Engineer to join our dynamic development team. This entry-level role offers an excellent opportunity to gain hands-on experience in software development, web applications, and cloud technologies. The ideal candidate will be passionate about coding, eager to learn, and capable of collaborating within an Agile environment. You will contribute to building and maintaining innovative solutions across various platforms, supporting our mission to deliver high-quality software products.

Duties

  • Assist in the design, development, and testing of web applications and software solutions using languages such as JavaScript, Java, C#, Python, and Ruby on Rails.
  • Contribute to front-end development tasks including UI/UX improvements and responsive design implementation.
  • Develop and optimize database queries using SQL, MySQL, and other relational database systems.
  • Collaborate with team members on application development projects utilizing version control tools like GitHub, SVN, and Git.
  • Support deployment processes on cloud platforms such as AWS and Azure, including configuring APIs and RESTful services.
  • Participate in Agile development cycles, including sprint planning, stand-ups, and code reviews.
  • Write clean, efficient code adhering to best practices for software development and application security.
  • Troubleshoot issues across different environments including Linux and Windows-based systems.
  • Document technical specifications and assist in maintaining existing codebases for continuous improvement.

Qualifications

  • Bachelor’s degree in Computer Science, Software Engineering, or a related field; relevant internship experience is a plus.
  • Basic knowledge of web development frameworks and front-end technologies such as HTML5, CSS3, JavaScript frameworks (React.js or Angular preferred).
  • Familiarity with programming languages including Java, C#, Python, C++, Ruby on Rails, or VBA.
  • Understanding of SQL databases such as MySQL or SQL Server; experience with database design is advantageous.
  • Exposure to cloud services like AWS or Azure along with API integration experience (REST APIs).
  • Knowledge of version control systems including GitHub, Git, SVN.
  • Experience working within Agile methodologies and collaborative tools.
  • Strong problem-solving skills with attention to detail and a willingness to learn new technologies. This position provides a comprehensive platform for growth within the software engineering field while contributing meaningfully to innovative projects across multiple technology stacks.
