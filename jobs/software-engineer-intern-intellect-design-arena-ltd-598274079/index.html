About Intellect Design Arena Ltd

Toronto-based software company with global clients.

Full Description

📌 Role: Software Engineer Intern

📍 Location: Toronto, Ontario, Canada (Onsite)

🎓 Experience: 0-1 years of professional experience

⏳ Internship Duration: 3 months (With potential for extension and conversion)

Key Responsibilities

Development & Coding

  • Design, develop, and maintain software applications under the guidance of senior engineers
  • Write clean, efficient, and well-documented code following best practices
  • Participate in code reviews and learn from feedback
  • Debug and fix bugs in existing applications

Collaboration & Learning

  • Work closely with cross-functional teams including product managers, designers, and QA engineers
  • Collaborate with team members to solve technical challenges
  • Contribute to technical discussions and brainstorming sessions

Quality & Testing

  • Write unit tests and participate in testing activities
  • Assist in identifying and resolving software defects
  • Documentation & Knowledge Sharing
  • Document code, processes, and technical decisions
  • Learn and adapt to new technologies and frameworks

Education

  • Bachelor's degree in Computer Science, Software Engineering, Information Technology, or related field

Technical Skills

  • Programming Languages: Proficiency in at least one programming language (Java, Python, JavaScript, C++, C#, etc.)
  • Basic Understanding of:
  • Object-Oriented Programming (OOP) concepts
  • Data structures and algorithms
  • Software development lifecycle (SDLC)
  • Version control systems (Git)

Familiarity with:

  • Web development (HTML, CSS, JavaScript) OR
  • Backend development OR Technical (Nice to Have)
  • Experience with cloud platforms (AWS, Azure, GCP)
  • Knowledge of databases (SQL or NoSQL)
  • Familiarity with RESTful APIs
  • Experience with testing frameworks
  • Experience with frontend frameworks (React, Angular, Vue) or backend frameworks (Spring, Django, Express)
