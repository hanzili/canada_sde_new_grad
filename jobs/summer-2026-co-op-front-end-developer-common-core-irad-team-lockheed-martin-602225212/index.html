About the role

Common Core is an inner source project within Lockheed Martin Canada centered around the CMS 330 product. The Common Core developer will work with a team dedicated to retooling an existing legacy codebase to allow code reuse, while still allowing program customization as needed. The developer will communicate design ideas to the product owner, development team, and other stakeholders using tools such as user flows, journey diagrams, interface mock-ups, wireframes, and prototypes. They will implement these designs using primarily Java. As an inner source project, the team‚Äôs primary stakeholders are internal developers, across multiple in-progress Programs within Lockheed Martin Canada.

Responsibilities :
