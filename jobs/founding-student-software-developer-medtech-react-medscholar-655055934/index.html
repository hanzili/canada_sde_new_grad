About the Role

Build React frontend, implement clinical case workflows

About MedScholar

MedTech education startup with exam-simulation platform

Full Description

I’m recruiting student developers interested in building a real MedTech product from the ground up .

I’m a practicing physician and medical educator developing an exam simulation platform for Canadian medical licensing exam preparation. The goal is to create an interactive web platform that helps medical trainees practice clinical reasoning in realistic structured exam environments.

This is not a classroom project. It’s an early-stage build toward a working education product used by real learners.

I’m looking for students in:

Computer Science

Software Engineering

Computer Engineering

Engineering Science

--especially those interested in healthcare, EdTech, AI-assisted learning tools, or startups.

What you’ll work on:

• React-based frontend development

• clinical case simulation workflows

• scoring + structured feedback systems

• user authentication architecture

• preparation for payment integration

• shaping the technical foundation of a real platform

You’ll be collaborating directly with me on product direction and implementation decisions.

This is ideal if you want:

• a serious portfolio project

• startup-style product-building experience

• exposure to healthcare technology development

• ownership-level contribution (not task-level coding)

• something you can talk about in future technical interviews

Strong fundamentals matter more than prior startup experience. React experience preferred.

Commitment

Flexible weekly collaboration time.

Compensation

Includes an initial $500 stipend , with the expectation of expanding compensation as the platform grows and reaches milestones.

If this sounds interesting, comment below or message me directly.

Happy to share more details about the platform and roadmap.
