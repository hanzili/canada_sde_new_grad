About Conversant Teamware Inc

Full Description

ABOUT CONVERSANT TEAMWARE

Conversant Teamware is an Alberta-based AI startup building the next generation of enterprise messaging — multi-threaded group chat with an AI layer that understands what your team needs before they ask. Our platform is protected by five granted patents covering AI-agent-assisted commerce, visual multi-threaded messaging, and knowledge graph intelligence.

Our market is oil and gas operations: field crew coordination, incident response, and in-chat procurement — where fast, structured communication saves time and money. You'll work directly with the CTO, touch real production code early, and grow fast.

Learn more: conversantlab.com

——————————————————————

THE ROLE

As our Junior Software Developer, you'll be embedded in the core product team from day one. You'll build and ship across our backend services, AI agent pipeline, and integration layer — with mentorship and clear ownership from the start.

Your responsibilities will include:

• Building and maintaining backend services in Node.js and Python

• Writing scripts, tooling, and data pipelines to support our graph and NLP workflows

• Integrating third-party APIs (LLMs, webhooks, messaging platforms)

• Writing unit tests and participating in code reviews

• Collaborating directly with senior developers and the founding team on architecture decisions

——————————————————————

REQUIRED SKILLS

• Node.js — REST APIs, async/await, basic WebSocket familiarity

• Python — scripting, data processing, working with libraries

• Git and version control — branching, pull requests, collaborative workflow

• SQL and basic databases — queries, schemas, data modelling fundamentals

——————————————————————

BONUS SKILLS (will accelerate your application)

• React / TypeScript — our web client stack

• Redis / Kafka — pub/sub and event streaming basics

• Graph databases — Neo4j or Cypher curiosity

• LLM / AI APIs — OpenAI or Anthropic SDK experience

• Docker / containers — local dev and basic Kubernetes

• WebSockets — real-time messaging fundamentals

——————————————————————

WHAT WE'RE LOOKING FOR

• 1–2 years of experience, or strong personal or academic projects — we care about what you've built

• Comfortable asking questions and owning tasks with guidance

• Genuine curiosity about AI, graph data, and real-time systems

• Location: Calgary or Edmonton

——————————————————————

WHAT YOU'LL GET

• Ground-floor opportunity at a patented AI platform startup

• Equity participation

• Exposure to graph AI, real-time systems, and enterprise SaaS — a rare junior résumé builder

• Flexible, remote-friendly environment with a Calgary-based team

——————————————————————

davidt@conversantlab.com

We review every application personally. No recruiters.
