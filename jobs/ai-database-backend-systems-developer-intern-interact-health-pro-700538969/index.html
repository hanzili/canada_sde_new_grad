Full Description

AI DATABASE & BACKEND SYSTEMS DEVELOPER INTERN

(Data Architecture & Automation)

About Us

Interact Health PRO (IHP) supports personal injury clients, healthcare providers, and legal professionals through AI-supported systems designed to improve rehabilitation coordination and medical documentation.

We are building database and automation systems that help structure client records, communication tracking, forms management, and workflow automation.

Role Overview

As an AI Database & Backend Systems Developer Intern, you will help develop and optimize the core infrastructure that powers IHP’s medical-legal coordination systems.

You will support:

  • Database architecture
  • Backend development
  • API integrations
  • Automation systems
  • Intelligent workflow structures

This role is ideal for students interested in backend systems, structured data, APIs, and AI-supported workflow automation.

Key Responsibilities

  • Design and optimize database structures
  • Support secure medical-legal data management
  • Develop APIs and backend integrations
  • Implement automation workflows using Airtable and Make.com
  • Improve database performance and scalability
  • Create documentation and system procedures
  • Support testing, QA, and deployment

AI Integration

  • Claude AI-assisted backend documentation
  • AI-assisted query optimization
  • Intelligent workflow automation
  • AI-enhanced correspondence systems

Technical Skills

  • SQL / MySQL
  • C# / ASP.NET
  • APIs and backend systems
  • Database architecture
  • GitHub

Assets

  • AWS exposure
  • Automation experience
  • Cloud infrastructure knowledge

Soft Skills & Professional Traits

  • Analytical thinker
  • Detail-oriented
  • Strong organizational skills
  • Problem-solving mindset
  • Team collaboration

Learning Outcomes

  • Build enterprise-level backend systems
  • Gain experience with healthcare-related data structures
  • Learn workflow automation systems
  • Develop AI-enhanced database workflows

﻿

Why Join Us?

Become part of a 25+ IHP Intern Team and collaborate closely with Web Developers, Full Stack App Developers, AWS Engineers, Business Analysts, IT & Security Networking, Medical-Health, Social Media, Video Editors, and Administrative Groups. Together, we are launching Interact Health PRO’s innovative platforms to transform healthcare, patient outcomes, and virtual care technology.
