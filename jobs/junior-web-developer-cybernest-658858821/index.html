About Cybernest

Full Description

Overview
We are seeking a skilled and innovative Web Developer to join our dynamic development team. The ideal candidate will be responsible for designing, developing, and maintaining high-quality web applications and websites. This role offers an exciting opportunity to work on diverse projects utilizing modern technologies and frameworks, contributing to the overall success of our digital solutions. Candidates should have a strong foundation in both front-end and back-end development, with experience in various programming languages, frameworks, and tools to deliver seamless user experiences and robust backend systems.

Responsibilities

  • Develop, test, and deploy scalable web applications using technologies such as HTML, CSS, JavaScript, Angular, React, and Vue.js.
  • Design responsive UI/UX interfaces with a focus on usability and visual appeal utilizing SCSS, Bootstrap, and UX best practices.
  • Build and maintain server-side components using frameworks like Spring Framework, Django, Ruby on Rails, ASP.NET, or Node.js.
  • Integrate third-party APIs and develop custom RESTful and SOAP web services for data exchange.
  • Manage databases including MySQL, PostgreSQL, MongoDB, SQL Server, and SQLite; optimize queries for performance.
  • Implement CI/CD pipelines using Jenkins, GitHub Actions, or Azure DevOps to streamline deployment processes.
  • Use version control systems such as Git or SVN for code management and collaboration.
  • Write unit tests with frameworks like Jasmine, Mocha, or Jest to ensure code quality and reliability.
  • Collaborate with cross-functional teams using Agile methodologies via tools like Jira and Trello.
  • Utilize containerization tools such as Docker and orchestration platforms like Kubernetes for deployment environments.
  • Maintain system architecture documentation and contribute to SDLC processes ensuring best practices are followed.
  • Support DevOps practices including automation with Ansible or Bash scripting on Linux environments.
  • Optimize websites for SEO performance and implement data visualization techniques for analytics dashboards.

Experience

  • Proven experience in full-stack web development with a portfolio of completed projects demonstrating expertise in front-end (HTML5, CSS3, JavaScript frameworks) and back-end (Java Spring Boot, .NET Core, Python Django).
  • Hands-on experience working with cloud platforms such as AWS or Azure for hosting and deployment purposes.
  • Familiarity with containerization (Docker), orchestration (Kubernetes), and continuous integration/continuous deployment (CI/CD) pipelines.
  • Strong understanding of system architecture principles including MVC patterns, RESTful APIs, GraphQL APIs, and microservices architecture.
  • Experience working within Agile/Scrum teams using project management tools like Jira or Trello is preferred.
  • Knowledge of e-commerce platforms (WordPress, WooCommerce) or CMS systems such as Drupal is a plus.
  • Ability to work effectively in Linux environments; familiarity with DevOps tools like Jenkins or Ansible is advantageous.
  • Excellent problem-solving skills combined with strong communication abilities to collaborate effectively across teams. This position offers an engaging environment where innovation is encouraged—ideal for developers eager to grow their skills while contributing to impactful projects across various industries including e-commerce, enterprise solutions, or digital media platforms.
