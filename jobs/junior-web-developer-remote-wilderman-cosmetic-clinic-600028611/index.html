Full Description

About Wilderman Cosmetic Clinic:
Wilderman Cosmetic Clinic is a healthcare organization specializing in cosmetic and medical services. We also manage digital platforms and internal websites to support our clinics and marketing initiatives.

Job Description
Wilderman Cosmetic Clinic is seeking a skilled and motivated Web Developer to join our team. This is a fully remote position which involves building, maintaining, and optimizing websites and web applications. Candidates must be located in Ontario, Canada, and able to work during Eastern Time business hours.

The ideal candidate will have experience in web development and be comfortable working with multiple technologies and tools. You will collaborate with our marketing and design teams to build, maintain, and optimize websites and web applications that are functional, user-friendly, and visually appealing.

Key Responsibilities:

  • Develop and maintain websites using PHP, JavaScript, HTML, and CSS
  • Customize and build WordPress themes and plugins
  • Optimize websites for performance, responsiveness, and security
  • Collaborated with the team to implement UX/UI improvements
  • Troubleshoot, debug, and resolve technical issues across the front and back end
  • Manage databases (MySQL) and server environments (cPanel)
  • Strong attention to detail and willingness to learn

Qualifications:

  • PHP, JavaScript, HTML, CSS
  • WordPress (theme and plugin development)
  • MySQL / Database Management
  • Web Performance Optimization and Debugging
  • Responsive Design / Mobile-First Development
  • UX/UI Implementation
  • Web Security Best Practices
  • WooCommerce
  • Server Management (cPanel, hosting environments)
  • Debugging and Troubleshooting
