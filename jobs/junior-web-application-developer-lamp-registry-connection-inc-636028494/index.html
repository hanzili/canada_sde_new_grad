Junior Web Application Developer (LAMP)

at Registry Connection Inc.

📍 Location Calgary, AB
📋 Details Full-time · $50k-$70k/yr
📅 Posted Mar 03, 2026

About the Role

Develop PHP web apps, fix bugs, implement features

About Registry Connection Inc.

Alberta-based registry software company.

Full Description

Registry Connection Inc. is a small Alberta based web application software company that specializes in solutions tailored to Alberta Registries and related industries.

Our web application software solutions include road test scheduling, point-of-sale, knowledge testing tools, wait/queue systems and other registry related software services.

We are looking for a junior LAMP developer for a permanent full-time salaried development position. Initial contract will be for a 3 month full-time probation, followed by permanent full-time employment, depending on review.

Please do not apply if you do not live in, or are not willing to relocate to Calgary, AB.

Must haves:

  • Reside in Calgary - This is a non-negotiable requirement for employment.
  • Ability to talk to developers and business clients alike, calmly and respectfully. Effective communication skills are an absolute must.
  • Linux experience (apache configurations/troubleshooting, common shell commands, log file management, etc.), as well as tools such as Putty and SSH.
  • PHP coding experience (multiple versions). Experience with out-of-the-box frameworks such as CakePHP or Laravel are an asset.
  • A solid understanding of the benefits of MVCs and OOP, and some experience with each.
  • MySQL/MariaDB experience, from both a GUI platform, and the ability to hand-bomb MySQL queries and update statements on the fly, and understand database and entity relationship diagrams.
  • JavaScript, jQuery, Bootstrap and CSS

Successful applicant should be able to use jQuery built-ins and libraries, but also have no issue with manual manipulation of the DOM with vanilla javascript when necessary.

  • Revision control experience (ie: git/bitbucket)

Nice to haves:

  • Android application development
  • Domain management experience (DNS records, hosting settings etc.)
  • System Management systems, such as Plesk, WHM+cPanel, and/or Webmin/Virtualmin
  • Node.js experience
  • Experience with graphical tools such as Photoshop
  • Basic video management experience (splicing, encoding)
  • Windows networking, roaming profiles, folder redirection, etc.
  • Comfort and familiarity using AI for coding. You will be asked to provide hand-coded work - this is not a vibe-coding environment.

Job Types: Full-time, Permanent

Pay: $50,000.00-$70,000.00 per year

Benefits:

  • Dental care
  • Extended health care
  • Vision care

Work Location: In person
