Description Become a Junior Systems Developer (New Grad) (10) with MPAC Big Data. Real Impact. MPAC is the largest property assessment jurisdiction in North America. Our Information Technology team builds and maintains platforms that power property data for every municipality in Ontario - open source tech, cloud computing, advanced analytics, and machine learning models. We tackle complex problems in an agile environment, and we move fast.

The Stack - Real tools. Real scale. From day one you are across a modern, enterprise-grade stack:

  • Front-end: React, TypeScript, HTML, CSS3
  • Back-end: Java, Rust, Node.js, Python, Linux
  • Data: Postgres, micro services, AJAX, SQL, Spark, JSON
  • Infrasturcture: Kubernetes, docker, AWS Cloud, CI/CD, Jira
  • Methods: Agile / Scrum, Rapid Prototyping

Not Your Regular First Job! Role In this role based in Pickering, Ontario, you'll work at the intersection of modern software development and enterprise-scale systems. What you can expect:

  • Exposure across the full stack and multiple technologies from day one
  • A team of skilled, collaborative engineers invested in your growth
  • Leadership that supports and backs you!
  • Open source tech, cloud computing, advanced analytics and machine learning
  • Competitive compensation, friendly and open environment where your work truly matters
  • Work that is genuinely consequential - municipalities in Ontario depend on what we build

Job Description What You Bring - Sharp. Curious. Recent Grad. About We don't just want someone who can write clean code - we want the person who has been knee-deep in a custom build, who's deployed their own Kubernetes cluster just to host a personal project, who is passionate about database indexing strategies for fun. What you will bring:

  • Recently completed a degree in Computer Science, Software Engineering, or related field
  • Front-end chops: React or Angular, JavaScript, TypeScript, HTML/CSS
  • Strong Object-Oriented Programming (OOP) fundamentals and solid data structure knowledge
  • Familiarity with Python, Rust, webservices, Hibernate
  • Can build dynamic single-page appications
  • Writes clean, documented code and uses version control like second nature
  • Communicates clearly
  • Self-directed and comfortable managing competing priorities
  • Keeps up with tech trends because you find it interesting

What you will do - Build Things That Matter!

  • Ship UI features and debug interfaces based on real technical requirements
  • Write high-quality, well-documented, tested code alongside senior engineers who want to mentor you
  • Support systems analysis on proposed enhancements - understand full impact before a line gets written
  • Build prototypes and proof-of-concepts: mock demos, performance tests, scalability checks
  • Participate in design reviews ensuring solutions meet functional and architectural standards
  • Document business problems and existing code - great engineers leave things better than they found them
