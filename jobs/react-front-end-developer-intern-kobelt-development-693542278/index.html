About Kobelt Development

Full Description

Overview

We are looking for a motivated Front End Developer Intern to assist with development and support of the YourRadar white-label platform.

This role is ideal for a student or junior developer who has working knowledge of React and wants real-world experience building and supporting SaaS business applications.

The successful candidate will work closely with our senior development team to help maintain, improve, and extend the user interface and front-end components of the platform.

Responsibilities

  • Assist with development and maintenance of React-based front-end applications
  • Support UI/UX improvements and white-label customization projects
  • Read, understand, and modify existing React components and application flows
  • Assist with debugging and troubleshooting front-end issues
  • Work with APIs and backend developers to integrate application features
  • Participate in testing and quality assurance of new features
  • Help document front-end functionality and development processes
  • Use Git for source control and collaboration
  • Participate in team discussions, planning, and code reviews
  • Learn and contribute to modern SaaS development practices

Skills & Qualifications

The successful candidate should demonstrate:

  • Basic to intermediate experience with React and JavaScript
  • Understanding of HTML, CSS, and responsive web design
  • Familiarity with REST APIs and JSON
  • Experience using Git or GitHub for version control
  • Strong problem-solving and troubleshooting abilities
  • Good communication skills and willingness to learn
  • Ability to work independently and ask questions when needed
  • Self-motivation and attention to detail

Nice to Have (Not Required)

  • Experience with TypeScript
  • Familiarity with Node.js or PHP
  • Exposure to Docker or containerized environments
  • Experience with UI component libraries
  • Understanding of SaaS applications or business software
  • Experience working on school, personal, or open-source projects

What You’ll Gain

  • Real-world experience working on a commercial SaaS platform
  • Mentorship from experienced developers
  • Exposure to modern web application architecture and deployment practices
  • Experience working with white-label software products
  • Opportunity to contribute meaningful code to production systems
