Full Description

Job Title: Junior Full Stack Developer

Location: Toronto, ON, Canada(hybrid)

Employment Type: Full-time

Rate: CAD70k-CAD90k/annum

About the Role:

Full-stack developer role requiring 1+ year experience in Node.js, React, and SQL, focusing on developing and maintaining scalable applications. Responsibilities include coding, testing, debugging, and contributing to automated builds while ensuring design compliance. The role involves collaborating in Agile teams, supporting documentation, and improving systems using new technologies. Strong problem-solving skills and ability to multitask are essential, with AWS and Python knowledge as added advantages.

Responsibilities

About the Role:

  • You will develop in multiple modules and architectures, according to development best practices and current development technologies
  • Contribution to automated build process
  • Systems documentation and knowledgebase management for support efficiencies
  • Ensure designs are in compliance with specifications
  • Design and conduct tests
  • Further develop current software
  • Support users by developing documentation and assistance tools
  • Support continuous improvement by investigating alternatives and new technologies and presenting these for architectural review.
  • Explore opportunities to add value to job accomplishments

Qualifications

About You:

  • 1 years Application Development experience
  • Full-stack developer with hands on experience on Nodejs and React
  • Experience with SQL
  • Nice to have Knowledge of AWS Cloud (IAM Roles, Pipelines, Cloud Stack Formation, etc...)
  • Comfortable working with Agile methodology and tools
  • Nice to have experience on Python and Pandas
  • Ability to prioritize and multi-task in order to meet deadlines
  • Can resolve technical issues through debugging, research and investigation
  • A team player, effective working independently and collaboratively
  • Reviewing code work for accuracy and functionality including peer code reviews
  • B.S. Degree in Computer Science, Computer Engineering or equivalent experience
