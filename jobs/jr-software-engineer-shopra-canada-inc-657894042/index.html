Junior Full Stack Developer

Location: Remote

Type: Full-Time

About Shopra

Shopra is building the operating system for beverage brands. We unify distributor, retailer, and digital data into one platform so beverage teams can clearly see how their products are performing in market.

We’re a small, fast-moving team working closely with real beverage operators. We’re looking for a Junior Full Stack Developer who’s excited to learn, ship real features, and grow alongside the platform.

What You’ll Do

  • Build and improve features across our Ruby on Rails + React stack
  • Work with product and design to turn requirements into working features
  • Write clean, maintainable code and learn best practices around testing
  • Help debug issues and improve performance over time
  • Participate in code reviews and team discussions
  • Learn how we deploy and operate the platform (Docker, cloud infrastructure)

Our Stack

  • Ruby on Rails (API)
  • React (TypeScript)
  • PostgreSQL
  • RSpec
  • Docker
  • GCP

What We’re Looking For

  • 1-3 years of experience with Ruby on Rails and React
  • Basic understanding of APIs and relational databases
  • Some experience with testing, or strong interest in learning
  • Comfortable in a fast-moving startup environment
  • Curious, dependable, and open to feedback
  • Bonus (not required): exposure to Docker or cloud platforms

Why Join Us?

  • Work on a real product used by beverage brands you recognize
  • Learn from experienced engineers and operators in the beverage industry
  • Grow your skills across backend, frontend, and infrastructure
  • Remote-friendly team that values learning, ownership, and shipping

Job Type: Full-time

Pay: $45,000.00-$55,000.00 per year
