Software Developer Intern

at Perseus Group, Constellation Software

📍 Location Markham, Ontario
📋 Details Intern · $24.91-$30.44/hr

About the Role

Build APIs, implement backend services, test.

About Perseus Group, Constellation Software

Canadian software group with diverse enterprise portfolio.

Full Description

THE POSITION The successful Software Development Intern works in our Markham Ontario office and reports to our Solutions Architect for the homebuilding industry’s leading-edge enterprise software product.

The Software Development Intern works in a Scrum team to build and maintain high performing, resilient and manageable web-based ERP software applications for homebuilders. Responsibilities include:

  • Design and implement RESTful and/or GraphQL APIs
  • Assist with API documentation using tools like Swagger/OpenAPI
  • Debug and optimize existing APIs for performance and reliability
  • Build backend services that integrate with databases, third-party services, and internal systems
  • Study and Understand our application's functionality, workflow, and design.
  • Collect, organize, and preprocess application knowledge
  • Document the process and provide recommendations for scaling and future enhancements

Qualifications And Skills

  • Be working from the Markham Ontario office
  • Understanding of software requirements
  • Demonstrated ability to leverage AI tools as a development aid, with a focus on learning, efficiency, and best engineering practices

  • Hands-on experience in Full Stack Development:

  • HTML5, TypeScript, JavaScript, Stylesheets

  • .NET
  • SQL Query
  • Knowledge in Object-Oriented Design
  • Knowledge in Test Driven Development
  • Ability to perform Unit Testing as well as Integration Testing
