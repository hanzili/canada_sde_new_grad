About the role

Internal Research and Development (IRAD) program is looking for a dynamic and detail-oriented student to join our team in support of our Ground Based Air Defence development team. Working alongside the team lead, quality assurance, and developers the candidate will build the human interfaces that control the mission planner tool that allows operators to plan and organize ground based air defence missions. The candidate has experience in developing user interfaces, creating mock-ups and prototypes to test design ideas through various user testing methods. The candidate will communicate design ideas to the team lead, development team, and other stakeholders using tools such as user flows, activity diagrams, interface mock-ups, and prototypes. They will implement these designs using React and typescript.

Responsibilities :

  • Create mock-ups, and interactive prototypes of potential UX/UI solutions
  • Create unit tests to ensure solutions work as expected
  • Develop responsive and accessible web interfaces using Hypertext Markup Language (HTML), Cascading Style Sheets (CSS) and using the React Front End and a Nest.js backend
  • Consume, define and integrate RESTful APIs endpoints to display dynamic data
  • Optimize applications for maximum speed and scalability
  • Use version control systems like Git for collaboration and project management
  • Write clean, reusable, and maintainable code
  • Conduct unit, integration, and end-to-end testing.

What you bring to the role

  • Current university or college student in a relevant field
  • Aptitude for programming and debugging
  • React, TypeScript, HTML, CSS, NestJS, Jest
  • Neutralionjs
  • Java, JavaFx, and Junit

  • Continuous Integration (CI)/Continuous Delivery (CD) pipelines experience in Jenkins.

Language Requirements in Quebec

Knowledge of the English language is required for this role as the position described in this posting requires the candidate to communicate every day with a virtual team outside Quebec as well as communicate with different members of the organization, clients and suppliers located outside of Canada. For engineering positions, the candidate must also be able to read or modify technical documents from our main clients and write technical documents for these same clients.
