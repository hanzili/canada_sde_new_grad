Full Description

Job Title: Junior Salesforce DevOps Engineer

Location: Toronto, ON (3x onsite a week)

Employment Type: Contract

Pay Rate: CAD$35 - $40/HR INC

Job Description We are looking for a Junior Salesforce DevOps Engineer (2–3 years) who will support daily release operations, metadata deployments, environment maintenance, and basic development tasks. This hybrid Admin‑Developer role involves managing pull requests, coordinating sandbox activities, assisting with DevOps pipelines, and developing Salesforce components using Apex, LWC, and integration concepts. The candidate will collaborate with senior developers, troubleshoot deployment issues, and maintain documentation.

Roles & Responsibilities

  • Deployment Operations: Manage daily pull requests, resolve metadata merge conflicts, and execute deployments between sandboxes.
  • Environment Management: Coordinate sandbox refreshes, manage data seeding, and ensure all environments remain synchronized.
  • Quality Support: Assist QA by running static code analysis (PMD/SonarQube) and automated unit tests within the CI/CD pipeline.
  • Troubleshooting: Perform root cause analysis for deployment issues and support developers with technical fixes.
  • Documentation: Maintain SOPs and technical release notes in Confluence or Jira.
  • Development & Customization: Develop code using Apex, Visualforce, and LWC to build custom business functionality.
  • System Configuration: Configure Salesforce objects, workflows, validation rules, and page layouts.
  • Support & Maintenance: Troubleshoot platform issues, support data migration tasks, and maintain system documentation.
  • Collaboration: Work closely with senior developers, technical architects, and business stakeholders to deliver enhancements.

Qualifications & Skills

  • Experience: 2–3 years of experience with the Salesforce platform and version control tools (Git/GitHub).
  • Technical Skills: Familiarity with Salesforce CLI and at least one DevOps tool (Flosum/AutoRABIT); working knowledge of Apex, JavaScript, HTML, and CSS.
  • Platform Knowledge: Good understanding of the Salesforce data model and declarative tools (Flows, Process Builder).
  • Additional Experience: 1–2 years in software development or Salesforce administration.
  • Soft Skills: Strong collaboration abilities, attention to detail, and proactive problem‑solving mindset.
  • Certifications: Salesforce Administrator and Platform Developer I certifications are highly desirable.
