Full Description

Per Vices is looking for a software engineer who can help us build Software Defined Radios (SDR) for mission critical systems. Our customers and products use high throughput (>1GBps), low latency (<200 Œº s), high performance software applications to implement various types of end applications.   • Experience with C or C++

Minimum Qualifications
